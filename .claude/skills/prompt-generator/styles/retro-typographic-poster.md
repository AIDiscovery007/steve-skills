# Retro Typographic Poster Style

## Style Overview

A bold vintage poster design style characterized by oversized retro typography as the dominant visual element, combined with silhouette figures and understated ironic messaging. Inspired by mid-century graphic design, cinema posters, and protest art aesthetics. Perfect for statement pieces, social commentary, brand campaigns, and artistic typography experiments.

**Visual Characteristics**:

- Bold oversized retro lettering as focal point
- Silhouette figures with expressive gestures
- Muted cool color palette (dusty blues, warm grays, faded ochres)
- Subtle floating text elements or crossed-out words
- Grainy vintage texture overlay
- Clean balanced composition despite bold elements
- Mid-century graphic design influence
- Cinema poster aesthetics
- Stylish irony and wit
- Strong typographic hierarchy

**Use Cases**:

- Social commentary posters
- Brand campaign visuals
- Music event / concert posters
- Film and cinema promotional art
- Editorial illustrations
- Statement wall art
- Protest / activism graphics
- Typography showcase pieces
- Vintage-inspired advertisements
- Book / album cover designs

## Standard Parameter Configuration

### Basic Configuration

```text
--ar 4:5 --stylize 80-150 --v 7
```

**Parameter Explanation**:

| Parameter | Recommended Value | Rationale |
|-----------|------------------|-----------|
| Aspect Ratio | `--ar 4:5` or `--ar 2:3` | Classic poster proportions |
| Stylization | `--stylize 80-150` | Balanced artistic interpretation for vintage feel |
| Version | `--v 7` | Better typography rendering and prompt following |
| Personalization | `--profile [code]` | Optional: apply personal aesthetic preferences |

### Parameter Adjustments for Different Scenarios

**Classic Poster (Vertical)**:

```text
--ar 2:3 --stylize 100 --v 7
```

- Traditional poster ratio
- Moderate stylization for vintage balance

**Square Format (Social Media)**:

```text
--ar 1:1 --stylize 120 --v 7
```

- Square for Instagram/social sharing
- Slightly higher stylization for impact

**Wide Format (Banner/Header)**:

```text
--ar 16:9 --stylize 80 --v 7
```

- Horizontal for web banners
- Lower stylization for cleaner text

**Film Poster Style**:

```text
--ar 27:40 --stylize 150 --v 7
```

- Classic movie poster ratio
- Higher stylization for cinematic feel

## Quick Usage Guide

### Step 1: Define Your Message

Choose impactful text for typography focus:
- Questions work well: "WHY?", "WHAT NOW?", "WHO CARES?"
- Statements: "ENOUGH", "THINK", "LISTEN"
- Contradictions: "LOUD SILENCE", "VISIBLE INVISIBLE"
- Commands: "WAKE UP", "PAY ATTENTION", "SAY SOMETHING"

### Step 2: Select Figure Treatment

Decide on silhouette approach:
- Expressive gestures (speaking/not speaking irony)
- Single dramatic figure
- Group/crowd silhouettes
- No figures (typography only)

### Step 3: Choose Color Palette

Select mood through color:
- Cool muted (sophisticated, contemporary)
- Warm vintage (nostalgic, inviting)
- Monochrome (dramatic, stark)
- Two-tone bold (impactful, graphic)

### Step 4: Define Era Reference

Pick aesthetic influence:
- Mid-century modern (clean, geometric)
- Cinema poster (dramatic, narrative)
- Protest art (bold, urgent)
- Swiss typography (grid, precision)

### Step 5: Set Texture Level

Choose texture intensity:
- Subtle grain (refined)
- Film grain (vintage photo)
- Halftone (print aesthetic)
- Distressed (aged, weathered)

### Step 6: Configure Parameters

Based on use case:
- Vertical poster → `--ar 2:3` or `--ar 4:5`
- Square social → `--ar 1:1`
- Film poster → `--ar 27:40`
- Keep `--stylize` between 80-150 for vintage balance

### Step 7: Fill Template

Insert your elements into the universal structure

## Common Adjustment Needs

### If Typography Isn't Prominent Enough

**Strengthen text focus**:
- Add: `typography as dominant element, text fills 60% of frame`
- Specify: `extra bold lettering, maximum visual weight`
- Include: `typography-first composition, all other elements secondary`

---

### If Too Modern Looking

**Increase vintage feel**:
- Add: `authentic vintage print aesthetic, analog production feel`
- Increase texture: `heavy film grain, visible paper texture, print imperfections`
- Specify era: `authentic 1960s poster production style`
- Lower color saturation: `desaturated colors, faded vintage tones`

---

### If Colors Too Bright

**Mute the palette**:
- Add: `desaturated color palette, muted tones, faded colors`
- Specify: `no bright colors, no high saturation, aged color quality`
- Use terms: `dusty`, `muted`, `faded`, `warm gray`

---

### If Composition Too Busy

**Simplify**:
- Reduce elements: `minimal elements, focus on typography and one supporting element`
- Add space: `generous negative space, clean uncluttered composition`
- Simplify background: `subtle background texture only, no floating elements`

---

### If Not Ironic/Witty Enough

**Increase conceptual tension**:
- Add contrast: `visual irony, contradiction between text and image`
- Include: `clever visual commentary, wry humor implied`
- Create paradox: `figures unable to do what text suggests`

## Important Notes

### Typography Best Practices

1. **Keep text short**: 1-4 words work best for maximum impact
2. **Use impactful words**: Questions, commands, or contradictions
3. **Consider readability**: Ensure text remains legible despite artistic treatment
4. **Test with --draft**: Iterate typography quickly before final render

### V7 Advantages for This Style

- Better text rendering for cleaner typography
- Improved prompt following for specific compositions
- Use `--draft` for rapid iteration on typography variations
- Positive descriptions work better than negatives
