# Ethereal Watercolor Portrait Style

## Style Overview

A delicate, traditional watercolor portrait style emphasizing transparency, soft bleeding edges, and romantic atmosphere. This style captures the essence of classical watercolor painting techniques while creating feminine, sensual, and ethereal character portraits. Perfect for game character fan art, romantic illustrations, and artistic portraits with an intimate mood.

**Visual Characteristics**:

- Transparent watercolor washes with soft bleeding edges
- Wet-on-wet technique for hair and fabric rendering
- Organic pigment dispersions and watercolor blooms
- Delicate skin with subtle color gradients
- Romantic warm lighting with soft glow effects
- No harsh outlines - forms defined by color transitions

**Use Cases**:

- Game character fan art (Genshin Impact, Wuthering Waves, etc.)
- Romantic portrait illustrations
- Feminine character art with intimate mood
- Art prints and poster designs
- Social media art posts (Pixiv, Twitter, etc.)

## Standard Parameter Configuration

### Recommended Parameters

```text
--ar 3:4 --stylize 350 --v 6
```

**Parameter Rationale**:

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| Aspect Ratio | `--ar 3:4` | Portrait orientation, ideal for half-body |
| Stylization | `--stylize 350` | High enough for artistic watercolor interpretation |
| Version | `--v 6` or `--v 7` | V6 for proven stability, V7 for better prompt following |

### Parameter Adjustments

**More Realistic Watercolor**:
```text
--stylize 250-300 --style raw
```
- Lower stylization for more authentic traditional watercolor look

**More Artistic/Stylized**:
```text
--stylize 400-500
```
- Higher stylization for more dramatic artistic interpretation

**Different Aspect Ratios**:

| Ratio | Use Case |
|-------|----------|
| `--ar 2:3` | Full body portraits |
| `--ar 3:4` | Half-body (recommended) |
| `--ar 4:5` | Close-up with some background |
| `--ar 1:1` | Square social media posts |
| `--ar 9:16` | Vertical phone wallpapers |

## Common Adjustment Needs

### If Watercolor Effect Is Too Subtle

**Enhance watercolor**:
```text
Add: dramatic watercolor bleeds, heavy pigment dispersions, visible paint flow, saturated color washes
Increase: --stylize to 400-450
```

### If Watercolor Effect Is Too Strong

**Tone down watercolor**:
```text
Use: subtle watercolor tint, controlled color washes, gentle wet-on-wet blending
Decrease: --stylize to 250-300
Add: --style raw
```

### If Skin Looks Too Flat

**Enhance skin rendering**:
```text
Add: luminous skin with subtle subsurface scattering, warm undertones, delicate color variations across features
Specify: soft blush on cheeks, gentle shadows under jaw
```

### If Fabric Looks Too Opaque

**Enhance transparency**:
```text
Emphasize: gossamer thin, translucent, light catching, see-through quality
Add: light passing through fabric, fabric sheerness
```

### If Mood Is Too Serious

**Lighten mood**:
```text
Change expression to: gentle playful smile, warm friendly gaze
Add: soft sparkles, cheerful atmosphere, bright color accents
Use brighter palette colors
```

### If Background Is Too Busy

**Simplify background**:
```text
Use: soft gradient background, minimal atmospheric elements, clean negative space
Remove: detailed environment descriptions
```

## Important Notes

### Do's

1. **Always define medium first**: Start with "Delicate watercolor portrait"
2. **Specify watercolor techniques**: Include wet-on-wet, bleeds, blooms for authenticity
3. **Use soft modifiers**: "gentle", "soft", "delicate", "subtle" throughout
4. **Match lighting to mood**: Warm lighting for intimate, cool for ethereal
5. **Limit color palette**: 3-4 colors plus accents for cohesion
6. **Include paper texture**: Essential for watercolor authenticity
7. **Describe skin carefully**: Use color gradients, not flat descriptions

### Don'ts

1. **Don't use harsh terms**: Avoid "sharp", "bold", "vivid", "striking"
2. **Don't over-detail clothing**: Keep fabric descriptions light and flowing
3. **Don't use photorealistic terms**: No "8K", "hyperrealistic", "photograph"
4. **Don't mix incompatible styles**: Avoid "digital art", "3D render" etc.
5. **Don't forget exclusions (V6)**: Include "no harsh outlines" etc.
6. **Don't stylize too low**: Below 250 may lose watercolor artistic quality
7. **Don't use too many colors**: More than 5 main colors loses watercolor subtlety
