# 少年漫画风格指南 (Shounen Style Guide)

少年漫画（Shounen Manga）是以10-18岁男性为主要目标读者的漫画类型，强调热血、友情、努力与胜利。

---

## 核心价值观

### 少年漫画三大支柱

```
     【友情】
        △
       ╱ ╲
      ╱   ╲
     ╱     ╲
    ▽───────▽
 【努力】 【胜利】
```

**友情 (Friendship)**: 伙伴的羁绊是力量源泉
**努力 (Effort)**: 不断超越自我
**胜利 (Victory)**: 正义终将获胜

### 主题公式

```
主角弱小 → 遭遇挑战 → 努力训练 → 伙伴支持 → 突破极限 → 战胜强敌
```

## 视觉风格特征

### 画风特点

| 元素 | 少年漫画风格 | 描述 |
|------|--------------|------|
| 线条 | 粗犷有力 | 强调动感和力量 |
| 眼睛 | 锐利型，较小 | 坚定、充满决心 |
| 表情 | 夸张戏剧化 | 情绪表达直接有力 |
| 身体 | 肌肉线条明显 | 运动感、力量感 |
| 动作 | 动态夸张 | 打破物理规则的张力 |
| 背景 | 简化/省略 | 突出角色和动作 |
| 对比 | 高对比度 | 黑白分明，冲击力强 |

### 效果线系统

**核心效果线**:

| 效果 | 日语 | 用途 | 提示词 |
|------|------|------|--------|
| 集中线 | 集中線 | 聚焦/冲击 | focus lines, radial lines |
| 速度线 | スピード線 | 运动/速度 | speed lines, motion lines |
| 能量线 | 気迫線 | 气势/能量 | energy lines, aura effect |
| 爆炸线 | 爆発線 | 爆炸/冲击 | explosion lines, impact burst |
| 震动线 | 振動線 | 震动/威压 | rumbling effect, tremor lines |

**视觉音效**:

| 音效 | 日语 | 含义 | 使用场景 |
|------|------|------|----------|
| ゴゴゴゴ | gogogo | 威压感 | 强者登场、威胁 |
| ドドドド | dododo | 震动/冲击 | 大地震动、强烈冲击 |
| バキッ | baki | 骨折/破碎 | 打击命中 |
| ドンッ | don | 重击 | 拳头命中 |
| ズバッ | zuba | 斩击 | 刀剑攻击 |

## 角色设计

### 主角原型

#### 热血型主角

```yaml
特点:
  - 正义感爆棚
  - 永不放弃
  - 重视伙伴
  - 有点笨但努力

外观:
  - 刺猬头/乱发
  - 锐利眼神
  - 精瘦肌肉型
  - 标志性饰品（伤疤/手环等）

表情库:
  - 决心脸（最常用）
  - 愤怒脸（保护伙伴时）
  - 惊讶脸（发现真相）
  - 友情感动脸
  - 搞笑脸（被打脸/饿肚子）
```

**代表**: 孙悟空、路飞、鸣人、炭治郎

#### 天才型主角

```yaml
特点:
  - 智力超群
  - 冷静分析
  - 有原则
  - 孤僻或傲慢

外观:
  - 整齐发型
  - 锐利/深邃眼神
  - 瘦削身材
  - 常戴眼镜或特殊服饰

表情库:
  - 冷静分析脸
  - 冷笑脸
  - 偶尔的真诚
  - 极少的惊讶（反转时刻）
```

**代表**: 夜神月、L、工藤新一

### 对手设计

#### 宿敌型

```yaml
与主角关系:
  - 实力相当或更强
  - 互相看不顺眼但内心尊重
  - 可能后期成为伙伴

外观差异化:
  - 与主角形成视觉对比
  - 主角热血 → 对手冷酷
  - 主角阳光 → 对手阴郁

表情库:
  - 傲慢脸
  - 挑衅脸
  - 认真战斗脸
  - 隐藏的尊重
```

**代表**: 贝吉塔、佐助、爆豪

### 伙伴设计

**职能分配**:

| 角色 | 功能 | 典型特征 |
|------|------|----------|
| 智力担当 | 战术/分析 | 眼镜、冷静 |
| 搞笑担当 | 调节气氛 | 夸张表情 |
| 力量担当 | 正面战斗 | 壮硕身材 |
| 治愈担当 | 治疗/支援 | 温和表情 |
| 神秘担当 | 信息/引导 | 面具/斗篷 |

## 分镜技法

### 动作场景节奏

**标准战斗序列**:

```
1. 对峙 (1-2格，大格)
   ↓ 紧张感建立
2. 起势 (1格)
   ↓ 动作预告
3. 攻击序列 (3-5格，小格快节奏)
   ↓ 速度线密集
4. 命中瞬间 (1格，大格，破格)
   ↓ 集中线爆发
5. 反应 (1-2格)
   ↓ 结果确认
```

### 鸟山明式动作分镜

**视线引导**: 角色动作方向引导阅读流

```
格1: 角色向右看 →
格2: 敌人从右出现 ←
格3: 双方对峙
格4: 主角向右冲锋 →→→
格5: [大格] 碰撞！
```

**时间感压缩**: 用小格快速推进

```
┌────┬────┬────┐
│ 出拳│ 躲闪│ 反击│  ← 3格1秒
└────┴────┴────┘
```

**空间感扩展**: 用大格强调关键动作

```
┌─────────────────┐
│                 │
│   决定性一击！   │  ← 大格定格
│                 │
└─────────────────┘
```

### 热血场景模板

**觉醒场景**:

```
格1: [大格] 主角陷入绝境
格2: [小格] 回忆伙伴的话
格3: [小格] 决心眼神特写
格4: [破格] 力量爆发！
格5: [见开き候补] 觉醒后的新形态
```

**友情场景**:

```
格1: 伙伴陷入危险
格2: 主角冲向伙伴
格3: [大格] 「我不会让你伤害他们！」
格4: 伙伴感动表情
```

## 风格检查清单

```markdown
□ 线条是否粗犷有力？
□ 动作是否夸张动感？
□ 效果线是否使用得当（集中线、速度线）？
□ 表情是否足够戏剧化？
□ 节奏是否有快慢对比？
□ 高潮场景是否给予足够版面？
□ 是否有热血的台词配合画面？
□ 背景处理是否突出角色？
```
