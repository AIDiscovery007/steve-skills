# 少女漫画风格指南 (Shoujo Style Guide)

少女漫画（Shoujo Manga）是以10-18岁女性为主要目标读者的漫画类型，强调恋爱、情感、成长与内心世界。

---

## 核心价值观

### 少女漫画主题

```
       【恋爱】
          ♡
         ╱ ╲
        ╱   ╲
       ╱     ╲
      ▽       ▽
   【成长】 【情感】
```

**恋爱 (Romance)**: 心动、告白、相互理解
**成长 (Growth)**: 自我发现、变得更好
**情感 (Emotion)**: 细腻的内心世界

### 叙事公式

```
平凡少女 → 命运相遇 → 心动萌生 → 误会/阻碍 → 情感成长 → 心意相通
```

## 视觉风格特征

### 画风特点

| 元素 | 少女漫画风格 | 描述 |
|------|--------------|------|
| 线条 | 纤细柔和 | 优雅、精致 |
| 眼睛 | 大而闪亮 | 多重高光，占脸部大比例 |
| 表情 | 细腻丰富 | 微妙情感变化 |
| 身体 | 纤细修长 | 理想化比例 |
| 服装 | 精致详细 | 蕾丝、蝴蝶结、花纹 |
| 背景 | 抽象情感化 | 花朵、星星、光效 |
| 氛围 | 柔和梦幻 | 柔光、模糊边缘 |

### 效果系统

**核心效果**:

| 效果 | 日语 | 用途 | 提示词 |
|------|------|------|--------|
| 闪闪发光 | キラキラ | 心动/美丽 | sparkles, glitter, shimmering |
| 花瓣 | 花びら | 浪漫/恋爱 | flower petals, floating petals |
| 泡泡 | バブル | 梦幻/柔和 | bubble effect, soft circles |
| 星星 | 星 | 希望/梦想 | stars, starlight, twinkling |
| 柔光 | ふわふわ | 温柔/回忆 | soft glow, ethereal light |
| 心形 | ハート | 心动/恋爱 | hearts, heart shapes |

**情感背景**:

| 情感 | 背景元素 | 描述 |
|------|----------|------|
| 心动 | 花朵+闪光 | 背景开满花，闪闪发光 |
| 悲伤 | 雨滴+灰暗 | 无背景或雨天 |
| 幸福 | 阳光+花瓣 | 温暖柔和的光 |
| 紧张 | 斜线阴影 | 网点密集 |
| 憧憬 | 星空+光芒 | 梦幻背景 |

## 角色设计

### 女主角原型

#### 平凡少女型

```yaml
特点:
  - 善良体贴
  - 外表普通但有魅力
  - 有点笨拙可爱
  - 内心坚强

外观:
  - 及肩或长发
  - 大而明亮的眼睛
  - 柔和的面部线条
  - 可爱的发饰

表情库:
  - 元气微笑
  - 害羞脸红（最常用）
  - 惊讶圆眼
  - 感动泪光
  - 生气（可爱型鼓腮）
```

**代表**: 真琴《水果篮子》、园子《邻家怪物同学》

#### 倔强少女型

```yaml
特点:
  - 独立自主
  - 不轻易服软
  - 有自己的梦想
  - 嘴硬心软

外观:
  - 短发或马尾
  - 坚定的眼神
  - 活泼的表情
  - 简洁的服装

表情库:
  - 坚定决心脸
  - 傲娇脸（嘴硬）
  - 意外害羞脸
  - 搞笑吐槽脸
```

**代表**: �的井杏《会长是女仆大人》

### 男主角原型

#### 王子型

```yaml
特点:
  - 外表完美
  - 傲慢但专一
  - 有温柔一面（反差萌）
  - 背景好（富二代/学霸）

外观:
  - 精致帅气
  - 锐利但柔和的眼睛
  - 完美发型
  - 高挑身材

表情库:
  - 傲慢冷酷脸
  - 突然温柔脸（反差）
  - 吃醋脸
  - 保护欲爆发脸
```

**代表**: 道明寺《花样男子》、会长《会长是女仆大人》

#### 温柔系

```yaml
特点:
  - 温柔体贴
  - 默默守护
  - 有隐藏的过去
  - 善解人意

外观:
  - 柔和的眼神
  - 温和的微笑
  - 干净整洁
  - 气质出众

表情库:
  - 温柔微笑（默认）
  - 认真表情
  - 隐藏悲伤
  - 偶尔的调皮
```

**代表**: 由希《水果篮子》、花沢類《花样男子》

### 闺蜜设计

```yaml
功能:
  - 女主的倾诉对象
  - 推动剧情（撮合/误会）
  - 调节气氛

典型设定:
  - 开朗活泼型（元气闺蜜）
  - 成熟大姐型（恋爱顾问）
  - 毒舌吐槽型（清醒旁观）
```

## 分镜技法

### 情感表达节奏

**心动场景序列**:

```
1. 日常场景 (1-2格)
   ↓ 铺垫
2. 偶然接触/对视 (1格)
   ↓ 触发
3. 心跳特写 (1格，花瓣/心形效果)
   ↓ 内心反应
4. 脸红特写 (1格，大格)
   ↓ 外在反应
5. 独白 (1格) 「我的心...跳得好快...」
```

### 情感背景技法

**替代具象背景**:

```
不画具体场景，用抽象元素表达情感:

格1: [花朵背景] 角色对话 → 恋爱氛围
格2: [黑暗背景] 角色独白 → 悲伤孤独
格3: [星空背景] 角色憧憬 → 梦想希望
格4: [空白背景] 角色震惊 → 时间停止感
```

### 特写运用

**眼睛特写**: 少女漫画标志性技法

```
┌─────────────────────┐
│                     │
│    ★ 闪亮眼睛 ★    │
│    包含复杂情感     │
│    多重高光        │
│                     │
└─────────────────────┘

用途: 表达复杂情感、关键情绪点
技法: 多重高光、睫毛细节、泪光
```

**嘴唇特写**: 告白/亲吻场景

```
┌─────────────┐
│             │
│    唇部     │ ← 暗示亲吻
│             │
└─────────────┘
```

## 页面布局特点

### 柔和边框

```
标准格:              少女漫画格:
┌──────────┐         ╭──────────╮
│          │         │          │
│          │   →     │          │
│          │         │          │
└──────────┘         ╰──────────╯
  直角边框              圆角边框

波浪边框（回忆/梦境）:
╭～～～～～～～～～╮
│                │
│    回忆内容    │
│                │
╰～～～～～～～～～╯
```

### 无边框构图

```
少女漫画经常省略格子边框:

        人物A                人物B
          ↓                    ↓
         ○                    ○
        /|\    花瓣飘落      /|\
        / \                  / \

无边框 + 抽象背景 = 梦幻氛围
```

### 装饰元素分布

```
┌─────────────────────────────┐
│ ✿           ✿              │
│      角色在中央            │ ← 花朵点缀
│ ✿                    ✿    │
│          ★               │ ← 闪光效果
│    ○    ○    ○          │ ← 气泡
│ ♡              ♡          │ ← 心形
└─────────────────────────────┘
```

## 经典场景模板

### 心动场景

```
格1: [日常] 女主在做某事
格2: [偶遇] 男主出现
格3: [特写] 眼神交汇
格4: [效果] 花瓣飘落 + 闪光
格5: [大格] 女主脸红 + 内心独白
     「他...好帅...」
```

### 告白场景

```
格1: [全景] 浪漫场所（屋顶/夕阳/樱花树下）
格2: [中景] 两人对峙
格3: [特写] 男主认真表情
格4: [大格] 「我喜欢你。」
格5: [女主反应] 震惊 → 眼含泪光
格6: [跨页候补] 拥抱/牵手
```

### 误会场景

```
格1: 女主看到男主和别人在一起
格2: [特写] 震惊眼睛
格3: [黑暗背景] 内心崩塌独白
格4: 转身离开（背影）
格5: [雨天] 独自哭泣

效果: 背景从花朵 → 黑暗 → 雨滴
```

### 和好场景

```
格1: 男主追上来
格2: 「听我解释！」
格3: 女主背对
格4: [大格] 男主从背后拥抱
格5: [特写] 两人表情
格6: [花瓣背景] 和好拥抱
```

## 少年vs少女对比速查

| 维度 | 少年漫画 | 少女漫画 |
|------|----------|----------|
| 线条 | 粗犷有力 | 纤细柔和 |
| 眼睛 | 锐利较小 | 大而闪亮 |
| 背景 | 简化省略 | 抽象情感化 |
| 效果 | 集中线速度线 | 花瓣闪光心形 |
| 节奏 | 快节奏动作 | 慢节奏情感 |
| 高潮 | 战斗觉醒 | 告白拥抱 |
| 边框 | 锐利倾斜 | 柔和圆角 |

## 风格检查清单

```markdown
□ 眼睛是否足够大且有多重高光？
□ 线条是否纤细柔和？
□ 是否使用了情感化抽象背景？
□ 花瓣/闪光等装饰元素是否得当？
□ 表情是否足够细腻丰富？
□ 边框是否使用了柔和处理？
□ 高潮情感场景是否给予足够版面？
□ 内心独白是否恰当表达情感？
```
