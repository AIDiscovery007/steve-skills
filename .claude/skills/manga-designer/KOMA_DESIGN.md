# 格子设计技巧 (コマ Design)

日本漫画的格子（コマ）设计是控制叙事节奏、引导读者视线、传达情感的核心技术。

---

## 格子基础

### 什么是コマ？

**コマ (Koma)** = 漫画中的单个画格/面板

```
一个标准页面:
┌─────────────────────────┐
│ ┌─────┐ ┌─────┐        │
│ │コマ1│ │コマ2│        │
│ └─────┘ └─────┘        │
│ ┌───────────────┐      │
│ │    コマ3      │      │
│ └───────────────┘      │
└─────────────────────────┘
```

### 格子的功能

| 功能 | 说明 | 技法 |
|------|------|------|
| 时间控制 | 格子 = 时间单位 | 大格=慢，小格=快 |
| 空间定义 | 决定画面范围 | 远景/近景/特写 |
| 情绪传达 | 格子形状影响感受 | 尖角=紧张，圆角=柔和 |
| 阅读引导 | 引导视线流动 | 位置、大小、形状 |
| 节奏控制 | 控制叙事快慢 | 密度、变化 |

## 格子大小与节奏

### 基本公式

```
格子大小 ∝ 重要程度 ∝ 停留时间

大格 → 重要场景 → 读者停留久 → 慢节奏
小格 → 次要/过渡 → 快速扫过 → 快节奏
```

### 大小分类

| 类型 | 占页比例 | 节奏 | 使用场景 |
|------|----------|------|----------|
| 特大格 | 50-100% | 极慢(停顿) | 高潮、震撼场景 |
| 大格 | 30-50% | 慢 | 重要场景、情感爆发 |
| 中格 | 15-30% | 中等 | 标准对话、一般场景 |
| 小格 | 5-15% | 快 | 快速动作、次要细节 |
| 微格 | <5% | 极快 | 瞬间反应、节奏加速 |

### 节奏模式

#### 渐快模式 (加速)
```
大格 → 中格 → 小格 → 小格 → 小格
 慢  →  中  →  快  →  快  → 快

用途: 动作场景、紧张升级
```

#### 渐慢模式 (减速)
```
小格 → 小格 → 中格 → 大格
 快  →  快  →  中  → 慢

用途: 高潮后余韵、情感沉淀
```

#### 对比模式 (张弛)
```
小格小格小格 → 大格 → 小格小格小格 → 大格
  快速推进   → 停顿 →   快速推进   → 停顿

用途: 紧张与释放交替、战斗节奏
```

#### 均匀模式 (稳定)
```
中格 → 中格 → 中格 → 中格
 中  →  中  →  中  →  中

用途: 日常对话、平稳叙事
注意: 过多会显得单调
```

## 破格技法（はみ出し）

### 什么是破格？

**破格 (Hamidashi)** = 角色或物体突破格子边界

```
正常格:                    破格:
┌──────────┐              ┌──────────┐
│          │              │          │
│    ○     │              │    ○     │ ← 角色
│   /|\    │              │   /|\    ├──→ 突破边界
│   / \    │              │   / \    │
└──────────┘              └────╱╲────┘
                               ╱  ╲
                               突出部分
```

### 破格类型

#### 1. 动作破格

```
用途: 强调动作力度
示例: 拳头突破边界、武器挥出

┌─────────────┐
│             │  ★
│  角色出拳   │══╗ ← 拳头突破
│             │  ║
└─────────────┘  ║
                 ↓
```

**提示词**: `breaking panel border, fist extending beyond frame, dynamic overflow`

#### 2. 角色破格

```
用途: 强调角色存在感、全身展示
示例: 角色登场、重要人物

┌──────────────┐
│   ○ 头      │
│  /|\ 身     │ ↑
│              │ │ 上部突破
├──────────────┤
│   / \ 腿    │ ↓
│              │   下部突破
└──────────────┘
```

**提示词**: `character overlapping panels, breaking frame boundaries, full body extending beyond panel`

#### 3. 情绪破格

```
用途: 强调情绪爆发
示例: 愤怒、惊讶、感动

      突破！
        ↓
┌──────╱╲──────┐
│     ╱  ╲     │
│    ○愤怒○    │ ← 脸部突破
│      w       │
└──────────────┘
```

**提示词**: `emotional outburst breaking frame, expression bursting panel borders`

#### 4. 效果破格

```
用途: 强调特效、能量
示例: 爆炸、能量波

        ※※※ ← 爆炸溢出
┌──────※※※──────┐
│     ◐    ◑     │
│  ═══════════   │ ← 能量波
└────═════════───┘
        ↓
    效果延伸
```

**提示词**: `explosion breaking panel, energy effects overflowing frame, dramatic effect extending beyond borders`

### 破格使用原则

| 原则 | 说明 |
|------|------|
| 适度 | 每页最多2-3处破格 |
| 目的明确 | 每次破格都要有理由 |
| 层次分明 | 破格部分叠加在其他格子上 |
| 重点突出 | 突破的部分是想强调的 |

## 特殊格子布局

### 见开き（跨页大图）

两页合并为一个画面：

```
←─────────── 左页 ───────────→←─────────── 右页 ───────────→
┌────────────────────────────┬────────────────────────────┐
│                            │                            │
│                            │                            │
│         超 大 画 面         │         超 大 画 面         │
│                            │                            │
│         (左半部)           │         (右半部)           │
│                            │                            │
│                            │                            │
└────────────────────────────┴────────────────────────────┘
                             ↑
                         装订线
```

**使用规则**:
- 短篇中最多1-2次
- 保留给最高潮场景
- 注意装订线位置（中央不放重要内容）
- 两页作为整体设计

**适用场景**:
- 史诗级战斗
- 震撼真相揭露
- 壮观场景展示
- 情感爆发顶点

**提示词**: `double page spread, epic manga scene, dramatic widescreen composition`

### 纵长格（縦ゴマ）

垂直方向的长条格子：

```
┌───┐
│   │
│   │
│ 纵 │
│ 长 │
│ 格 │
│   │
│   │
└───┘
```

**特点**: 诗意、垂直运动感
**用途**: 下落、上升、角色全身、情感抒发

**风格对比**:
- 大岛弓子风格（少女漫画）: 多用纵格，诗意浪漫
- 小�的ユキ风格: 更多横格，电影感

**提示词**: `vertical manga panel, falling motion, full body vertical composition`

### 横长格（横ゴマ）

水平方向的长条格子：

```
┌─────────────────────────────────┐
│            横长格               │
└─────────────────────────────────┘
```

**特点**: 电影感、全景感
**用途**: 环境展示、追逐场景、对峙画面

**提示词**: `wide horizontal panel, cinematic manga composition, panoramic view`

### 多层格

格子内再分割：

```
┌────────────────────┐
│  ┌───┬───┐        │
│  │ A │ B │   C    │
│  ├───┴───┤        │
│  │   D   │        │
│  └───────┘        │
└────────────────────┘
```

**用途**: 同一时间多视角、快速反应、内心独白
**注意**: 不要过度使用，保持可读性

## 效果线系统

### 集中线（集中線）

从画面边缘向中心汇聚的线条：

```
        ╲│╱
       ──◎──  ← 焦点
        ╱│╲
```

**类型**:

| 类型 | 效果 | 用途 |
|------|------|------|
| 细线密集 | 紧张、速度 | 快速动作 |
| 粗线稀疏 | 冲击、震撼 | 重击、爆发 |
| 不规则 | 混乱、疯狂 | 情绪失控 |

**提示词**: `focus lines, radial lines, converging lines, dramatic radial blur`

### 速度线（スピード線）

表现运动方向的平行线：

```
════════════════→  方向
════════════════→
═══◎═══════════→  移动物体
════════════════→
════════════════→
```

**提示词**: `speed lines, motion lines, action lines, horizontal motion blur`

### 背景效果

#### 闪光效果（フラッシュ）

```
      ╲ │ ╱
     ── ★ ──
      ╱ │ ╲
```

**用途**: 震惊、灵感、觉醒
**提示词**: `flash effect, dramatic lighting burst, epiphany moment`

#### 实心闪光（ベタフラッシュ）

```
██╲ │ ╱██
███ ★ ███
██╱ │ ╲██
```

**用途**: 极度震撼、死亡、绝望
**提示词**: `solid black with white flash, dramatic high contrast, stark black and white`

#### 网点背景（スクリーントーン）

```
░░░░░░░░░░░░
░░░░░░░░░░░░
░░░░░░░░░░░░
```

**用途**: 阴影、质感、氛围
**密度变化**: 深→浅 表现光影过渡

**提示词**: `screentone, halftone pattern, dot shading, manga tones`

## 少女漫画特有效果

### 花瓣效果

```
  🌸      🌸
     🌸
  🌸    🌸
        🌸
```

**用途**: 浪漫、心动、美丽场景
**提示词**: `flower petals, floating petals, cherry blossoms, romantic atmosphere`

### 闪闪效果（キラキラ）

```
  ✦   ✧
    ✦
  ✧   ✦
```

**用途**: 心动、美丽、少女情怀
**提示词**: `sparkles, glitter, shimmering effect, shoujo sparkle`

### 气泡效果

```
  ○  ○
    ○
  ○    ○
```

**用途**: 梦幻、柔和、回忆
**提示词**: `bubble effect, soft circles, dreamy bubbles`

### 抽象情感背景

```
不画具体背景，用抽象图案表达情感:
- 花朵 → 恋爱/美丽
- 心形 → 心动
- 黑暗 → 悲伤/绝望
- 闪光 → 希望/决心
```

**提示词**: `abstract emotional background, shoujo manga style, symbolic background`

## 格子排列模式

### 标准2×4网格

日漫最常见的基础布局：

```
┌─────────┬─────────┐
│    1    │    2    │
├─────────┼─────────┤
│    3    │    4    │
├─────────┼─────────┤
│    5    │    6    │
├─────────┼─────────┤
│    7    │    8    │
└─────────┴─────────┘
```

**变化**: 合并/分割格子创造变化

### 动态变化布局

打破规则增加视觉兴趣：

```
┌───────────────────┐
│        1          │  ← 大格开场
├─────────┬─────────┤
│    2    │    3    │
├─────┬───┴───┬─────┤
│  4  │   5   │  6  │  ← 快节奏
├─────┴───────┴─────┤
│        7          │  ← 大格收尾
└───────────────────┘
```

### 翻页结构

**奇数页（右页）**:
- 阅读起点
- 放置"接上文"内容
- 可以有较复杂布局

**偶数页（左页）**:
- 阅读终点
- 右下角放悬念（めくり）
- 为翻页做铺垫

## 格子设计检查清单

```markdown
## 单页检查

□ 格子数量是否适合场景类型？（对话4-6，动作5-8）
□ 大小变化是否支持节奏？
□ RTL阅读流是否顺畅？
□ 右下角是否有翻页钩子（めくり）？
□ 破格使用是否适度且有意义？

## 整体检查

□ 起承转结是否有对应的格子节奏变化？
□ 高潮场景是否有足够大的格子？
□ 是否有视觉单调的连续页面？
□ 效果线使用是否统一风格？
```
