## 详细分镜

### 第1页 - 平凡少女的早晨

**页面功能**: 起·女主登场
**格子数**: 5格
**节奏**: 慢

```
┌─────────────────────────────────┐
│  格1 (大格 40%) - 建立镜头      │
│  春天，樱花飘落的上学路        │
│  雨晴的背影                    │
├────────────────┬────────────────┤
│  格2 (中格)    │  格3 (中格)    │
│  侧脸微笑      │  手握笔记本    │
├────────────────┴────────────────┤
│  格4 (中格) - 独白              │
│  「我叫苏雨晴...是个普通的     │
│  高二生」                       │
├─────────────────────────────────┤
│  格5 (中格) - 学校门口          │
│  效果: 花瓣飘落                 │
└─────────────────────────────────┘
```

#### 格1 (大格)
- **镜头**: 远景→中景
- **内容**: 春天的上学路，樱花飘落，雨晴的背影走向学校
- **对话**: 独白「春天...新学期开始了」
- **效果**: 樱花花瓣、柔和光线
- **提示词**: `shoujo manga opening scene, spring morning, cherry blossoms falling, schoolgirl walking from behind, school bag, gentle atmosphere, soft lighting, black and white, delicate linework --ar 16:9 --v 7`

#### 格2 (中格)
- **镜头**: 侧面特写
- **内容**: 雨晴侧脸，温柔地看着樱花
- **效果**: 闪光点缀
- **表情**: 温和微笑

#### 格3 (中格)
- **镜头**: 特写
- **内容**: 手握着一个小笔记本
- **对话**: 独白「这是我的诗集...虽然从没给人看过」

#### 格4 (中格)
- **镜头**: 正面中景
- **内容**: 雨晴对着读者介绍自己
- **对话**: 「我叫苏雨晴，高二生」「不太擅长...和人说话」
- **效果**: 粉色发卡特写

#### 格5 (中格)
- **镜头**: 远景
- **内容**: 学校门口，学生们走入
- **对话**: 独白「每天都一样...平凡的日子」
- **效果**: 花瓣、日常氛围
