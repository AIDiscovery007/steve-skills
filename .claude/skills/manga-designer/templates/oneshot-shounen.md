# 少年漫画読み切り模板 (Shounen Oneshot Template)

読み切り（Yomikiri）是指单话完结的漫画作品，通常用于杂志投稿、新人赛或连载前试水。

---

## 基本规格

### 页数范围

| 目标 | 页数 | 说明 |
|------|------|------|
| 新人赛 | 16-31页 | 週刊少年JUMP手�的�的奖标准 |
| 杂志投稿 | 32-45页 | 增刊投稿 |
| 同人/自主 | 16-24页 | 自由创作 |

**本模板以 16页 为基准**

### 格子预算

| 页码 | 阶段 | 格数(估) | 说明 |
|------|------|----------|------|
| 1-2 | 起 | 8-12 | 开篇吸引 |
| 3-6 | 承 | 20-30 | 铺垫展开 |
| 7-12 | 転 | 30-40 | 高潮战斗 |
| 13-16 | 結 | 15-20 | 结局余韵 |
| **总计** | | **70-100格** | |

---

## 结构模板

### 三幕式结构 (16页版)

```
【第一幕: 起】1-3页 (~20%)
├── P1: 冲击开场 / 动作开场
├── P2: 角色介绍 / 世界观展示
└── P3: 事件触发 / 冲突建立

【第二幕: 承】4-10页 (~45%)
├── P4-5: 冲突展开 / 对手登场
├── P6-7: 第一次交锋 / 挫折
├── P8-9: 低谷 / 觉悟
└── P10: 反击开始

【第三幕: 転・結】11-16页 (~35%)
├── P11-13: 最终决战 (高潮)
├── P14: 胜利 / 解决
└── P15-16: 余韵 / 新的开始
```

## 逐页分镜指南

### 第1页: 冲击开场

**目标**: 在3秒内抓住读者

**推荐技法**:
- 动作开场（战斗中）
- 悬念开场（神秘场景）
- 角色开场（主角帅气登场）

**格子分配**:
```
┌─────────────────────────┐
│                         │
│      大格 (70%)         │  ← 冲击画面
│      主角帅气登场       │
│                         │
├─────────────┬───────────┤
│    小格     │   小格    │  ← 补充信息
│    环境     │   反应    │
└─────────────┴───────────┘
```

**ネーム示例**:
```markdown
### P1 - 冲击开场

格1 (大格70%):
- 镜头: 仰视
- 内容: 主角站在废墟中，背对读者，面前是敌人
- 效果: 集中线，威压感
- [无对话]

格2 (小格15%):
- 镜头: 远景
- 内容: 被破坏的城市

格3 (小格15%):
- 镜头: 特写
- 内容: 主角握紧拳头
- 对话: 「我不会让你得逞的...」
```

### 第2-3页: 角色与冲突建立

**目标**: 让读者了解主角、关心主角

**要素检查**:
- [ ] 主角的目标/动机
- [ ] 主角的弱点/困境
- [ ] 主要冲突是什么
- [ ] 敌人/障碍是什么

**格子分配 (P2)**:
```
┌─────────────────────────┐
│   中格 - 主角日常/背景  │
├───────────┬─────────────┤
│   主角    │   对话/     │
│   互动    │   说明      │
├───────────┴─────────────┤
│   中格 - 触发事件       │
├─────────────────────────┤
│   悬念 - 威胁出现       │  ← めくり
└─────────────────────────┘
```

### 第4-7页: 第一次对决（挫折）

**目标**: 展示敌人强大，主角陷入危机

**节奏**: 快→更快→停顿（挫折）

**格子分配建议**:
- P4-5: 战斗展开（小格快节奏）
- P6: 主角被压制（大格强调危机）
- P7: 最低点（转折准备）

**ネーム示例 (P6 危机)**:
```markdown
### P6 - 危机时刻

格1 (大格50%):
- 镜头: 俯视
- 内容: 主角倒地，敌人俯视
- 对话: 敌人「就这种程度吗？」
- 效果: 阴影覆盖主角

格2 (中格):
- 镜头: 特写
- 内容: 主角眼神（不屈）
- [无对话]

格3 (小格) [めくり]:
- 镜头: 特写
- 内容: 主角手指动了一下
- 对话: (内心)「还...没有...」
```

### 第8-10页: 觉醒与反击

**目标**: 主角找到力量源泉，开始反击

**典型触发**:
- 回忆伙伴的话
- 想起要保护的人
- 觉醒隐藏力量
- 找到敌人弱点

**关键技法**:
- 回忆用波浪边框
- 觉醒用破格+能量效果
- 反击开始用速度线

**ネーム示例 (P9 觉醒)**:
```markdown
### P9 - 觉醒

格1 (回忆格):
╭～～～～～～～～～～╮
│ 伙伴: 「我相信你！」│
╰～～～～～～～～～～╯

格2 (特写):
- 内容: 主角眼睛睁开（决心）
- 效果: 闪光

格3 (大格 破格):
- 镜头: 全身
- 内容: 主角站起，能量爆发
- 效果: 集中线+能量线
- 对话: 「我...绝不会放弃！」
- 破格: 能量突破格子边界
```

### 第11-13页: 最终决战（高潮）

**目标**: 整个故事最精彩的部分

**版面分配**:
- 考虑使用见开き（跨页）
- 最重要的一击给最大格子
- 小格密集→大格爆发

**节奏模式**:
```
快速攻防 (小格×6) →
对峙 (中格) →
最终一击 (见开き/大格) →
结果 (中格)
```

**ネーム示例 (P12-13 见开き)**:
```markdown
### P12-13 - 最终一击 (跨页)

[整个跨页为一格]
- 镜头: 动态构图
- 内容: 主角与敌人最终碰撞
- 左页: 主角冲锋，拳头向前
- 右页: 敌人迎战
- 中央: 能量碰撞爆发
- 效果: 集中线、速度线、爆炸效果
- 对话: 主角「这是我的...全部力量！！」
- 注意: 避免在装订线放重要内容
```

### 第14页: 胜利

**目标**: 战斗结束，敌人倒下

**格子分配**:
```
┌─────────────────────────┐
│                         │
│   大格 - 敌人倒下       │
│   或主角胜利姿态       │
│                         │
├─────────────────────────┤
│   中格 - 反应           │
├─────────────────────────┤
│   中格 - 余波           │
└─────────────────────────┘
```

### 第15-16页: 余韵与结尾

**目标**: 给读者留下余韵，暗示未来

**结尾类型**:
- **希望型**: 新的冒险开始
- **温馨型**: 伙伴团聚
- **开放型**: 暗示更大的故事
- **循环型**: 回到开场的变体

**格子分配 (P16)**:
```
┌─────────────────────────┐
│   角色们的日常 / 未来   │
├───────────┬─────────────┤
│  温馨     │   对话      │
│  互动     │   或独白    │
├───────────┴─────────────┤
│                         │
│   大格 - 结束画面       │
│   主角望向未来/走向远方 │
│   「我的故事，才刚开始」│
│                         │
└─────────────────────────┘
```

## 整页生成指南

本节用于将企划阶段设计的每页内容合并为一张完整的漫画页图片生成。

### 整页生成原理

少年漫画整页生成将一页内所有格子合并为一个结构化提示词：
- 使用 `panel layout` 描述格子布局和数量
- 按阅读顺序描述每格内容
- 少年漫画特有：强调动作、速度线、冲击效果

### 推荐宽高比

| 页面类型 | 宽高比 | 说明 |
|----------|--------|------|
| 标准漫画页 | `--ar 2:3` | 最接近杂志比例 |
| 竖版强调 | `--ar 9:16` | 纵向动作强调 |
| 见开き跨页 | `--ar 16:9` 或 `--ar 2:1` | 横向展开 |

### 整页提示词基础结构

```
shounen manga page, [X] panel dynamic layout,
action manga style, dramatic composition,

panel 1 ([位置]): [镜头] [角色] [动作] [表情],
panel 2 ([位置]): [镜头] [角色] [动作] [效果],
panel 3 ([位置]): [镜头] [角色] [动作] [表情],
...

speed lines, impact effects, high contrast,
black and white manga, professional shounen quality
--ar 2:3 --v 7 --style raw
```

### 战斗页整页模板

#### 攻防交锋页 (5格快节奏)

```
shounen manga page, 5 panel dynamic action layout,
intense battle sequence, fast-paced combat,

panel 1 (top right): wide shot, two fighters facing off, tense atmosphere,
panel 2 (top left): [主角] launching attack, speed lines, determined expression,
panel 3 (middle large): clash moment, both attacks colliding, impact burst effect,
panel 4 (bottom right): [敌人] counterattack, aggressive pose, motion blur,
panel 5 (bottom left): [主角] dodging or blocking, strained expression, speed lines,

dynamic action composition, heavy speed lines throughout,
black and white manga, high contrast, impact effects
--ar 2:3 --v 7 --style raw
```

#### 劣势危机页 (4格压迫感)

```
shounen manga page, 4 panel layout,
protagonist in danger, overwhelming enemy,

panel 1 (top large): [敌人] dominant pose from low angle, intimidating, shadow over hero,
panel 2 (middle right): [主角] being pushed back, struggling, damage visible,
panel 3 (middle left): close-up [主角] pained expression, sweat, determination fading,
panel 4 (bottom large): [主角] on knees or falling, [敌人] looming, hopeless atmosphere,

oppressive atmosphere, dramatic shadows,
black and white manga, heavy screentone for darkness
--ar 2:3 --v 7 --style raw
```

#### 反击开始页 (5格转折)

```
shounen manga page, 5 panel layout,
turning point, hero rising,

panel 1 (top): [主角] down, seemingly defeated, darkness,
panel 2 (middle right): flashback small panel, memory of [重要的人/话], soft border,
panel 3 (middle left): [主角] eyes opening, spark of determination,
panel 4 (middle large): [主角] starting to stand, energy gathering, light breaking through,
panel 5 (bottom): [主角] fully standing, new resolve, power awakening, dramatic lighting,

hope emerging from despair, increasing intensity,
black and white manga, contrast from dark to light
--ar 2:3 --v 7 --style raw
```

### 角色一致性技巧

少年漫画角色需要在战斗中保持可辨识：

```
核心视觉特征重复:
- 发型: "young man with spiky black hair"
- 服装: "character in torn black coat"
- 特征道具: "hero wielding large sword"

每个panel都包含关键特征:
panel 1: young man with spiky black hair charging,
panel 2: same spiky black haired hero attacking,
panel 3: spiky black haired protagonist taking hit,
```

### 整页生成 vs 逐格生成选择

| 页面类型 | 推荐方式 | 原因 |
|----------|----------|------|
| 快速战斗页 | 整页生成 | 统一动态感 |
| 觉醒/变身页 | 整页生成 | 能量效果连贯 |
| 见开き跨页 | 整页生成 | 必须统一构图 |
| 表情特写页 | 逐格生成 | 需要精细控制 |
| 复杂多角色页 | 逐格生成 | 各角色需单独调整 |

## 常见问题

| 问题 | 症状 | 解决 |
|------|------|------|
| 开篇无力 | 读者没兴趣 | 用动作/悬念开场 |
| 战斗无聊 | 只是打打打 | 加入策略/情感 |
| 觉醒太假 | 突然变强 | 提前铺垫线索 |
| 结局仓促 | 赢得太快 | 延长最终战斗 |
| 节奏平淡 | 没有起伏 | 增加低谷和高潮对比 |
