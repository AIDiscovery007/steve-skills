# 少女漫画読み切り模板 (Shoujo Oneshot Template)

少女漫画的読み切り（Yomikiri）专注于恋爱、情感与成长主题，节奏相对柔和，重视内心描写。

---

## 基本规格

### 页数范围

| 目标 | 页数 | 说明 |
|------|------|------|
| 新人赛 | 20-40页 | 少女杂志新人奖 |
| 杂志投稿 | 24-50页 | 花与梦等投稿 |
| 同人/自主 | 16-24页 | 自由创作 |

**本模板以 24页 为基准**

### 格子预算

| 页码 | 阶段 | 格数(估) | 说明 |
|------|------|----------|------|
| 1-4 | 起 | 16-20 | 建立世界与角色 |
| 5-12 | 承 | 35-45 | 感情发展与铺垫 |
| 13-20 | 転 | 35-45 | 误会、冲突、高潮 |
| 21-24 | 結 | 16-20 | 和好与余韵 |
| **总计** | | **100-130格** | |

---

## 结构模板

### 恋爱故事结构 (24页版)

```
【第一幕: 相遇】1-6页 (~25%)
├── P1-2: 女主日常 / 性格展示
├── P3-4: 命运的相遇！
└── P5-6: 第一印象 / 心动萌芽

【第二幕: 靠近】7-14页 (~35%)
├── P7-8: 再次相遇 / 交集增加
├── P9-10: 了解对方 / 好感升温
├── P11-12: 心动确认 / 独白「我...喜欢上他了」
└── P13-14: 接近成功...但！

【第三幕: 误会】15-20页 (~25%)
├── P15-16: 误会发生！(看到他和别人)
├── P17-18: 心碎 / 逃避
└── P19-20: 真相 / 追逐

【第四幕: 告白】21-24页 (~15%)
├── P21-22: 告白场景
└── P23-24: 甜蜜结局 / 余韵
```

## 逐页分镜指南

### 第1-2页: 女主登场

**目标**: 让读者认识并喜欢女主

**展示要素**:
- 性格特点（善良/元气/内向...）
- 日常生活
- 内心独白

**格子分配 (P1)**:
```
┌─────────────────────────┐
│   女主日常场景          │
│   (上学路上/房间/学校)  │
├─────────────────────────┤
│   中格 - 独白介绍       │
│   「我叫XX，是个普通的高中生」│
├───────────┬─────────────┤
│  日常     │   性格      │
│  细节     │   展示      │
└───────────┴─────────────┘
```

**ネーム示例**:
```markdown
### P1 - 女主登场

格1 (大格40%):
- 镜头: 远景→中景
- 内容: 樱花飘落的上学路
- 女主背影走向学校
- 独白:「春天，新学期开始了...」

格2 (中格):
- 镜头: 中景侧面
- 内容: 女主微笑着看樱花
- 独白:「我叫苏雨晴，高二生」
- 效果: 樱花花瓣点缀

格3 (小格):
- 镜头: 特写
- 内容: 手握书包带
- 表情: 带着期待

格4 (中格):
- 镜头: 正面
- 内容: 女主对着读者微笑
- 独白:「今年...会有什么特别的事发生吗？」
- 效果: 轻微闪光
```

### 第3-4页: 命运的相遇

**目标**: 男女主相遇，建立第一印象

**经典相遇方式**:
- 转角相撞
- 被男主帮助
- 意外接触（伞、笔记本）
- 在特殊场所相遇

**关键技法**:
- 男主登场用大格+闪光效果
- 时间停滞感
- 女主心跳特写

**ネーム示例 (P3 相遇)**:
```markdown
### P3 - 命运的相遇

格1 (中格):
- 内容: 女主转角奔跑（要迟到了！）
- 对话:「糟糕糟糕！要迟到了！」

格2 (小格):
- 内容: 撞上某人
- 效果: 碰撞感

格3 (大格50% - 关键！):
- 镜头: 仰视
- 内容: 男主第一次登场
- 效果: 闪光背景、花瓣飘落、时间静止感
- 男主:「你没事吧？」
- [男主的帅气画面！]

格4 (特写):
- 内容: 女主睁大的眼睛
- 效果: 心跳、脸红
- 独白:（这个人...）
```

### 第5-6页: 第一印象与心动萌芽

**目标**: 建立对男主的印象，种下心动种子

**展示男主**:
- 温柔/冷酷（反差预告）
- 某个触动人心的行为
- 留下悬念（他是谁？）

### 第7-12页: 感情升温

**目标**: 两人交集增加，女主确认心意

**经典场景**:
- 再次偶遇
- 被安排在一起（同桌/搭档）
- 共同经历某事
- 发现对方的另一面

**内心描写技法**:
```
场景描述 + 独白框:
┌─────────────────────────┐
│  画面: 男主帮助她       │
│                         │
│    ┌────────────────┐  │
│    │ 他的手...好温暖...│ ← 独白
│    └────────────────┘  │
└─────────────────────────┘
```

**心动确认场景 (P11-12)**:
```markdown
### P11 - 心动确认

格1 (中格):
- 内容: 两人日常互动

格2 (特写):
- 内容: 男主微笑
- 效果: 闪光

格3 (大格):
- 内容: 女主呆住，脸红
- 心跳效果: ドキドキ
- 背景: 花朵绽放

格4 (特写):
- 内容: 女主捂住胸口
- 独白:「我的心...跳得好快...」

格5 (大格):
- 内容: 女主独自一人，脸红
- 独白:「我...喜欢上他了...」
- 效果: 大量花瓣、闪光
```

### 第13-14页: 关系进展...但！

**目标**: 让读者觉得快要成功了，然后设置阻碍

**技法**: 希望→绝望的落差

```
P13: 两人关系更近了！（希望）
P14最后一格: 但是...！（绝望伏笔）
→ 翻页钩子（めくり）
```

### 第15-18页: 误会与心碎

**目标**: 情感冲突的高潮

**经典误会**:
- 看到男主和别的女生在一起
- 听到片面的对话
- 第三者的介入
- 自己的自卑作祟

**视觉技法**:
- 背景从花朵→黑暗
- 眼睛失去高光
- 雨天场景
- 独自哭泣

**ネーム示例 (P15 误会发生)**:
```markdown
### P15 - 误会

格1 (中格):
- 内容: 女主开心地走向某处
- 独白:「今天要告诉他...」

格2 (中格):
- 内容: 看到男主和学姐在一起
- 画面定格

格3 (特写):
- 内容: 女主震惊的眼睛
- 效果: 瞳孔收缩、时间停止

格4 (大格):
- 内容: 学姐挽着男主的手臂
- 学姐:「学弟，你真温柔～」
- 效果: 黑暗阴影笼罩画面边缘

格5 [めくり]:
- 内容: 女主的书包掉落
- [无对话]
- 下一页: 逃跑/崩溃
```

### 第19-20页: 真相与追逐

**目标**: 误会解开的契机，男主追女主

**经典展开**:
- 男主发现女主误会
- 解释失败
- 雨中追逐
- 关键信息揭露

### 第21-22页: 告白场景

**目标**: 全作最甜蜜的高潮！

**经典告白场景**:
- 夕阳屋顶
- 雨中告白
- 樱花树下
- 特别的地点

**格子分配**:
```
┌─────────────────────────┐
│   场景建立 - 浪漫地点   │
├───────────┬─────────────┤
│  两人     │   对峙      │
│  站定     │   紧张      │
├───────────┴─────────────┤
│                         │
│   大格 - 「我喜欢你。」 │
│   男主认真表情          │
│                         │
├─────────────────────────┤
│   女主反应 - 泪目/幸福  │
└─────────────────────────┘
```

**ネーム示例 (P22 告白)**:
```markdown
### P22 - 告白

格1 (大格50%):
- 镜头: 中景
- 内容: 男主认真地看着女主
- 对话:「苏雨晴，我喜欢你。」
- 效果: 背景花瓣飘落、闪光
- 氛围: 时间静止

格2 (特写):
- 内容: 女主眼睛含泪
- 效果: 泪光闪烁

格3 (大格):
- 内容: 女主扑向男主
- 对话:「我...我也喜欢你！」
- 效果: 大量花瓣、心形
```

### 第23-24页: 甜蜜结局

**目标**: 糖分满载，让读者满足

**结尾类型**:
- 牵手/拥抱画面
- 两人的未来暗示
- 甜蜜日常一幕
- 接吻（可选）

**最后一格**:
```
┌─────────────────────────┐
│                         │
│   两人牵手走向未来      │
│   或 甜蜜对视           │
│                         │
│   独白:「这就是我的     │
│         恋爱故事的开始」│
│                         │
│   [完]                  │
└─────────────────────────┘
```

## 整页生成指南

本节用于将企划阶段设计的每页内容合并为一张完整的漫画页图片生成。

### 整页生成原理

少女漫画整页生成将一页内所有格子合并为一个结构化提示词：
- 使用 `panel layout` 描述格子布局和数量
- 按阅读顺序描述每格内容
- 少女漫画特有：花瓣、闪光、情感氛围、内心独白空间

### 推荐宽高比

| 页面类型 | 宽高比 | 说明 |
|----------|--------|------|
| 标准漫画页 | `--ar 2:3` | 最接近杂志比例 |
| 竖版情感页 | `--ar 9:16` | 纵向情感强调 |
| 宽幅浪漫页 | `--ar 3:4` | 稍宽的情感构图 |

### 整页提示词基础结构

```
shoujo manga page, [X] panel layout,
romantic [情感类型] atmosphere,

panel 1 ([位置]): [镜头] [角色] [动作] [表情],
panel 2 ([位置]): [镜头] [角色] [动作] [表情],
panel 3 ([位置]): [镜头] [角色] [动作] [表情],
...

[少女漫画效果: 花瓣/闪光/心形],
black and white manga, delicate linework, emotional screentone
--ar 2:3 --v 7
```

### 少女漫画装饰元素关键词

在整页生成中融入少女漫画特有的装饰元素：

| 情感 | 装饰元素关键词 |
|------|---------------|
| 心动 | `sparkles, floating flower petals, soft glow, dreamy atmosphere` |
| 害羞 | `blush marks, steam effect, heart symbols, embarrassed atmosphere` |
| 悲伤 | `rain drops, dark vignette, tears, melancholic atmosphere` |
| 幸福 | `abundant flower petals, bright sparkles, hearts floating, joyful radiance` |
| 紧张 | `dramatic shadows, heartbeat effect, tense atmosphere` |
| 浪漫 | `cherry blossoms, sunset glow, romantic lighting, intimate atmosphere` |

### 角色一致性技巧

少女漫画角色需要保持可爱和情感连贯：

```
女主核心特征重复:
- 发型: "young girl with long wavy brown hair"
- 眼睛: "large expressive eyes"
- 服装: "girl in school uniform with ribbon"

男主核心特征重复:
- 发型: "handsome young man with neat black hair"
- 特点: "tall, gentle expression"
- 服装: "boy in school uniform blazer"

每个panel保持特征:
panel 1: young girl with long wavy brown hair walking,
panel 2: same wavy brown haired girl blushing,
panel 3: close-up of wavy brown haired girl's expressive eyes,
```
