# 同人志模板 (Doujinshi Template)

同人志是由个人或小团体自主创作、印刷的漫画作品，可以是原创或二次创作。

---

## 基本规格

### 页数范围

| 类型 | 页数 | 说明 |
|------|------|------|
| 薄本 | 8-16页 | 小故事/插画集 |
| 标准本 | 20-32页 | 常见规格 |
| 厚本 | 40-60页 | 长篇故事 |

**本模板以 16页 为基准**

### 尺寸规格

| 规格 | 尺寸 | 说明 |
|------|------|------|
| B5 | 182×257mm | 标准同人志 |
| A5 | 148×210mm | 小本/便携 |
| A4 | 210×297mm | 大本/画集 |

---

## 同人志类型

### 原创同人 (Original)

完全原创的故事和角色：

```
特点:
- 世界观自由
- 角色设计自由
- 不受原作限制
- 需要更多世界观建立
```

### 二次创作 (Fanwork)

基于现有作品的创作：

```
特点:
- 角色已有人气
- 读者有背景知识
- 需尊重原作设定
- 可专注于"如果..."场景

常见类型:
- CP同人 (配对)
- 日常向 (slice of life)
- 平行世界 (AU)
- 续写/补完
```

## 同人志结构

### 原创同人结构 (16页)

```
【封面】封面 + 封底 (不计入)
【扉页】P1: 标题页/角色介绍
【正文】P2-14: 主要故事 (13页)
【结尾】P15: 结局
【后记】P16: 作者后记/预告

格子总数: 约50-70格
```

### 二次创作结构 (16页)

```
【封面】封面 + 封底
【扉页】P1: 标题/场景设定
【正文】P2-14: 同人故事
【结尾】P15: 甜蜜/搞笑结尾
【后记】P16: 后记/感谢
```

### 原创短篇模板 (P2-15)

```

- 世界观快速建立
- 主角登场
- 问题/事件提出

```

## 同人志特有技法

### 1. 快速世界观建立

同人志页数有限，需快速建立设定：

```markdown
技法A: 文字说明页
┌─────────────────────────────┐
│  在某个魔法学院...          │
│  主角小明是一名新生...      │
└─────────────────────────────┘

技法B: 开场独白
格1:「这里是XXX学院」
格2:「我是这里的学生」
格3: 学院全景

技法C: 默认读者已知 (二次创作)
直接开始故事，不解释背景
```

### 2. 精炼对话

每格对话控制在1-2句：

```
❌ 冗长:
「你知道吗，其实我一直以来都想告诉你一件事，
那就是从我们第一次见面的时候开始...」

✅ 精炼:
「我...一直想告诉你」
「什么？」
「我喜欢你」
```

### 3. 留白技法

用留白加强情感：

```
┌─────────────────────────────┐
│                             │
│                             │
│         [沉默的两人]        │
│                             │
│                             │
└─────────────────────────────┘
无对话，只有氛围
```

### 4. 二创暗语

二次创作常用省略技法：

```
不需要解释的元素:
- 角色外观 (读者已知)
- 基本人设 (读者已知)
- 世界观 (读者已知)

需要说明的元素:
- 本同人的时间点
- 平行世界设定
- 特殊前提
```

## 印刷注意事项

### 出血与安全区

```
┌─────────────────────────────┐
│←3mm出血                     │
│  ┌─────────────────────┐   │
│  │←5mm安全区            │   │
│  │                      │   │
│  │   [正文内容区域]     │   │
│  │                      │   │
│  │                 安全区→│   │
│  └─────────────────────┘   │
│                     出血→   │
└─────────────────────────────┘

重要内容不要放在:
- 边缘3mm (会被裁切)
- 边缘8mm (安全区外)
- 装订侧 (会被遮挡)
```

### 页码规划

```
同人志页码 (骑马钉装订):

物理页 → 内容
1 (封面外) → 封面
2 (封面里) → 可空白或广告
3 → P1 扉页
4 → P2 正文开始
...
N-1 → 最后一页正文
N (封底里) → 后记或空白
N+1 (封底) → 封底
```

### 总页数要求

骑马钉装订必须是4的倍数：
- 8页、12页、16页、20页、24页...

## 整页生成指南

本节用于将企划阶段设计的每页内容合并为一张完整的漫画页图片生成。

### 整页生成原理

同人志整页生成将一页内所有格子合并为一个结构化提示词：
- 使用 `panel layout` 描述格子布局和数量
- 按阅读顺序（日本漫画：右→左，上→下）描述每格内容
- 同人志特有：注重角色魅力和CP互动

### 推荐宽高比

| 尺寸规格 | 宽高比 | 说明 |
|----------|--------|------|
| B5 (标准同人志) | `--ar 5:7` | 最接近182×257mm |
| A5 (小本) | `--ar 5:7` | 148×210mm |
| 正方形变体 | `--ar 4:5` | 留白较多 |

### 整页提示词基础结构

```
manga doujinshi page, [X] panel layout,
[原创/二次创作] [类型] style,

panel 1 ([位置]): [镜头] [角色] [动作] [表情],
panel 2 ([位置]): [镜头] [角色] [动作] [表情],
panel 3 ([位置]): [镜头] [角色] [动作] [表情],
...

[情感氛围], [效果类型],
black and white manga, professional doujinshi quality
--ar 5:7 --v 7
```

### 角色一致性技巧

同人志需要在多页保持角色一致：

**核心特征重复法**:
```
每个panel描述中包含：
- 发型: "young man with messy silver hair"
- 特征配饰: "character with red scarf"
- 服装: "girl in school uniform with ribbon"

示例:
panel 1: young man with messy silver hair looking at window,
panel 2: same young man with messy silver hair turning around,
panel 3: close-up of young man with messy silver hair, surprised expression,
```

**二次创作角色描述**:
```
不要用原作角色名，改用外观描述:
❌ "Gojo Satoru talking"
✅ "tall man with white spiky hair and black blindfold speaking"
```

### 整页生成工作流建议

```
1. 企划阶段
   └── 逐格详细设计（保持不变）

2. 整页生成阶段
   ├── 封面：单独生成，高质量
   ├── 正文页：按页整页生成
   └── 关键页（告白/冲突）：可考虑逐格生成后合成

3. 检查与调整
   └── 检查角色一致性，必要时重新生成
```

## 检查清单

### 企划阶段

```markdown
□ 故事能否在页数内讲完？
□ 是否有明确的开始和结束？
□ 封面构思是否吸引人？
□ 页数是否为4的倍数？
```

### 分镜阶段

```markdown
□ 每页信息量是否适中？
□ 对话是否精炼？
□ 重要场景是否有足够版面？
□ 阅读节奏是否流畅？
```

### 印刷前

```markdown
□ 出血线是否正确设置？
□ 安全区内没有重要内容被切掉？
□ 页码顺序是否正确？
□ 分辨率是否足够 (300dpi以上)？
□ 封面封底文字是否清晰？
```
