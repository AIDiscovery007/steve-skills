# 连载章节模板 (Chapter Template)

连载漫画的单话章节模板，适用于週刊或月刊连载的单回内容设计。

---

## 基本规格

### 页数标准

| 刊物类型 | 标准页数 | 说明 |
|----------|----------|------|
| 週刊连载 | 17-19页 | 週刊少年JUMP等 |
| 隔週连载 | 20-24页 | 部分杂志 |
| 月刊连载 | 30-50页 | 月刊杂志 |

**本模板以週刊标准 18页 为基准**

### 章节类型

| 类型 | 功能 | 特点 |
|------|------|------|
| 开篇章 | 故事开始 | 类似読み切り，需完整世界观建立 |
| 推进章 | 日常推进 | 承上启下，铺垫伏笔 |
| 战斗章 | 战斗为主 | 动作密集，节奏快 |
| 高潮章 | 重大转折 | 震撼场面，情感爆发 |
| 过渡章 | 休整过渡 | 日常、修炼、情感 |
| 结局章 | 篇章结束 | 收束剧情，开启新篇 |

---

## 通用章节结构

### 三幕式结构 (18页)

```
【开篇】1-3页 (~17%)
├── P1: 承接上话 / 冲击开场
├── P2: 状况确认 / 问题明确
└── P3: 本话目标确立

【展开】4-12页 (~50%)
├── P4-6: 第一阶段 (尝试/对抗)
├── P7-9: 第二阶段 (困难/升级)
└── P10-12: 第三阶段 (转折点)

【收尾】13-18页 (~33%)
├── P13-15: 高潮 / 本话最精彩部分
├── P16-17: 结果 / 余波
└── P18: 悬念 / 下话预告
```

## 各类型章节模板

### 战斗章模板

**适用**: 少年漫画对战、Boss战

**节奏**: 快→更快→爆发→余波

```
P1-2: 战斗进行中 / 对峙
P3-4: 第一波攻防
P5-7: 敌人强势 / 我方劣势
P8-10: 转机出现 / 新技能 / 援军
P11-14: 最终对决
P15-16: 胜负分晓
P17-18: 余波 + 下一个敌人暗示
```

**格子分配**:
- 对话格: 30% (少)
- 动作格: 50% (多)
- 特写格: 20%

### 情感章模板

**适用**: 角色过去、羁绊深化

**节奏**: 慢→情感累积→爆发→余韵

```
P1-2: 现状铺垫
P3-6: 回忆开始 (波浪边框)
P7-10: 过去的痛苦/羁绊
P11-14: 情感爆发 / 理解达成
P15-17: 和解 / 成长
P18: 回到现实 + 新的决心
```

**格子分配**:
- 对话格: 40%
- 情感格: 40%
- 动作格: 20%

### 日常章模板

**适用**: 战斗间隙、角色互动

**节奏**: 轻松舒缓，偶有小高潮

```
P1-2: 日常场景开始
P3-6: 角色互动 A
P7-10: 角色互动 B (可能有小事件)
P11-14: 轻松高潮 (搞笑/温馨)
P15-17: 收束日常
P18: 暗示下一篇章 (悬念)
```

**格子分配**:
- 对话格: 50%
- 搞笑格: 30%
- 铺垫格: 20%

### 转折章模板

**适用**: 重大剧情转折、真相揭露

**节奏**: 铺垫→积累→爆炸

```
P1-3: 看似平常的开始
P4-7: 异常出现 / 伏笔回收
P8-12: 真相逐渐揭露
P13-16: 真相炸弹！(见开き候补)
P17-18: 角色反应 + 超级悬念
```

## 连载特有技法

### 1. 承上启下

**上话回顾 (P1)**:
```markdown
### P1 - 承接上话

技法选择:
A. 直接续接: 从上话最后一幕继续
B. 结果呈现: 展示上话战斗的结果
C. 时间跳跃: 「三天后...」
D. 回顾小格: 小格回顾关键画面后继续
```

**下话预告 (P18)**:
```markdown
### P18 - 悬念结尾

技法选择:
A. 新敌人登场剪影
B. 关键台词:「接下来...才是真正的开始」
C. 危机暗示: 某处的阴谋
D. 角色决心:「我要去XX！」
```

### 2. 多线叙事

连载可以同时推进多条故事线：

```
主线 (70%): 主角的冒险/战斗
副线A (15%): 其他角色的动向
副线B (15%): 敌人的计划

页面分配示例 (18页):
P1-3: 主线开场
P4-6: 主线推进
P7-8: 切换副线A
P9-12: 回到主线
P13-14: 切换副线B (伏笔)
P15-17: 主线高潮
P18: 副线A结果 + 悬念
```

### 3. 伏笔管理

连载需要跟踪伏笔：

```markdown
## 伏笔追踪表

| 伏笔 | 埋设章节 | 状态 | 计划回收 |
|------|----------|------|----------|
| 神秘戒指 | 第3话 | 未解 | 第15话 |
| 主角父亲 | 第1话 | 部分 | 第20话 |
| 敌人真实身份 | 第7话 | 未解 | 第12话 |
```

### 4. 人气管理

週刊连载需要关注读者反馈：

```
人气要素检查:
□ 本话有没有主角帅气画面？
□ 有没有情感共鸣点？
□ 有没有惊喜/反转？
□ 结尾悬念够不够强？
□ 是否有人气角色出场？
```

## 週刊/月刊差异

### 週刊连载 (17-19页)

**特点**:
- 节奏更快
- 每话必须有亮点
- 悬念更强调
- 对话更精炼

**禁忌**:
- 整话铺垫无高潮
- 过多说明文字
- 节奏太慢

### 月刊连载 (30-50页)

**特点**:
- 可以更详细铺陈
- 一话可完成小故事
- 战斗可更完整
- 情感可更细腻

**优势**:
- 更多版面给大场面
- 多用见开き
- 角色刻画更深

### 週刊 18页 格子预算

| 页码 | 功能 | 格数 | 备注 |
|------|------|------|------|

| 8-11 | 展开B | 16-20 | 第二阶段 |
| 12-15 | 高潮 | 16-20 | 本话亮点 |
| 16-17 | 收尾 | 8-10 | 结果处理 |
| 18 | 悬念 | 3-4 | 下话预告 |
| **总计** | | **70-90** | |

## 整页生成指南

本节用于将企划阶段设计的每页内容合并为一张完整的漫画页图片生成。

### 整页生成原理

连载章节整页生成将一页内所有格子合并为一个结构化提示词：
- 使用 `panel layout` 描述格子布局和数量
- 按阅读顺序描述每格内容
- 连载特有：需要考虑承上启下、悬念设置

### 推荐宽高比

| 页面类型 | 宽高比 | 说明 |
|----------|--------|------|
| 标准週刊页 | `--ar 2:3` | 最接近杂志比例 |
| 竖版强调 | `--ar 9:16` | 纵向戏剧效果 |
| 见开き跨页 | `--ar 16:9` 或 `--ar 2:1` | 横向展开 |

### 整页提示词基础结构

```
manga chapter page, [X] panel layout,
[章节类型: action/emotional/daily] chapter style,

panel 1 ([位置]): [镜头] [角色] [动作] [表情],
panel 2 ([位置]): [镜头] [角色] [动作] [效果],
panel 3 ([位置]): [镜头] [角色] [动作] [表情],
...

[效果类型], professional serialization quality,
black and white manga, clean panel borders
--ar 2:3 --v 7 --style raw
```

### 角色一致性技巧 (连载特化)

连载漫画需要长期保持角色一致：

```
建立角色视觉关键词档案:
主角: "young man with spiky black hair, scar on left cheek, wearing torn black coat"
反派: "tall figure with long silver hair, red eyes, dark armor"

在每页中重复核心特征:
panel 1: young man with spiky black hair and scar attacking,
panel 2: same scarred black-haired hero dodging,
panel 3: spiky black-haired protagonist with torn coat preparing,
```
