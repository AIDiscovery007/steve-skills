# 故事结构框架 (Story Structure)

本文档详细解释故事结构的设计方法，是 SKILL.md Phase 1 的深度参考。

---

## 结构类型详解

### 1. 起承転結 (Kishotenketsu) - 日式四幕

日本传统的四幕结构，源自汉诗和俳句，被广泛应用于漫画。

```
┌─────────────────────────────────────────────────────┐
│  起 (Ki)      │  承 (Sho)     │  転 (Ten)    │  結 (Ketsu)  │
│  Introduction │  Development  │  Twist       │  Conclusion  │
│  15-20%       │  25-30%       │  35-40%      │  15-20%      │
└─────────────────────────────────────────────────────┘
```

**起 (Ki) - 建立**:
- 功能: 建立世界观，介绍主角，展示日常
- 目标: 让读者理解"正常"是什么样
- 技巧: 用具体场景代替解说
- 常见错误: 过度解释、信息堆积

**承 (Sho) - 展开**:
- 功能: 引入事件，展开故事，升级冲突
- 目标: 建立期待，深化角色关系
- 技巧: 适度加速，埋设伏笔
- 常见错误: 节奏拖沓、冲突不升级

**転 (Ten) - 转折**:
- 功能: 故事最大转折，高潮场景
- 目标: 打破所有期待，情感爆发
- 技巧: 大格子、跨页、特殊技法
- 常见错误: 转折太弱、预期之内

**結 (Ketsu) - 收束**:
- 功能: 解决冲突，建立新平衡
- 目标: 给读者满足感，留有余韵
- 技巧: 呼应开头，暗示未来
- 常见错误: 太仓促、解释过度

**适用场景**:
- 日常系作品
- 情感驱动的故事
- 不以冲突为核心的叙事
- 短篇读み切り

### 2. 三幕式结构 (Three-Act Structure) - 西式

好莱坞经典结构，强调冲突和角色成长。

```
┌───────────────────────────────────────────────────────────┐
│     ACT 1     │            ACT 2            │    ACT 3    │
│     Setup     │        Confrontation        │  Resolution │
│     25%       │            50%              │     25%     │
│               │    ┌─────────┬─────────┐    │             │
│               │    │  Rise   │  Fall   │    │             │
│               │    │   to    │  from   │    │             │
│               │    │ Midpoint│ Midpoint│    │             │
└───────────────────────────────────────────────────────────┘
```

**Act 1 - Setup (建立)**:
```
Ordinary World: 展示主角的日常生活
    ↓
Inciting Incident: 打破平衡的事件
    ↓
Debate: 主角的犹豫和选择
    ↓
Plot Point 1: 主角决定行动，进入新世界
```

**Act 2 - Confrontation (对抗)**:
```
New World: 适应新环境，结识盟友/敌人
    ↓
Fun & Games: 探索新可能
    ↓
MIDPOINT: 重大转折，stakes升级
    ↓
Bad Guys Close In: 形势恶化
    ↓
All Is Lost: 最低点
    ↓
Dark Night of the Soul: 内心绝望
```

**Act 3 - Resolution (解决)**:
```
Plot Point 2: 发现解决之道
    ↓
Finale: 最终对决
    ↓
New Equilibrium: 建立新平衡
```

**适用场景**:
- 冒险类作品
- 成长类故事
- 有明确对手的情节
- 动作/战斗漫画

---

### 3. 混合结构

复杂故事可以结合两种结构：

```
框架: 三幕式
├─ Act 1: 起 + 承的前半
├─ Act 2: 承的后半 + 転
└─ Act 3: 結

优点:
- 西式的冲突驱动
- 日式的转折美学
- 灵活的节奏控制
```

## 高潮优先设计 (クライマックス先行)

### 设计原则

**为什么要高潮优先？**
- 读者记住的是高潮，不是铺垫
- 从高潮倒推，铺垫更有目的性
- 避免"为铺垫而铺垫"的问题

### 高潮场景设计清单

```markdown
## 高潮场景检查清单

### 视觉冲击 (10分)
□ 这个画面能让读者停下来吗？
□ 能用一个画面概括吗？
□ 适合用什么技法？(跨页/破格/大格)
□ 分镜师看到会兴奋吗？

评分: ___/10

### 情感爆发 (10分)
□ 角色的情绪达到顶点了吗？
□ 读者能产生情感共鸣吗？
□ 这个情感是铺垫了的吗？
□ 爆发是自然的还是强行的？

评分: ___/10

### 主题呼应 (10分)
□ 这里体现了核心主题吗？
□ 主角的成长在这里显现吗？
□ 是否回应了开头提出的问题？
□ 读完会让人思考吗？

评分: ___/10

### 叙事满足 (10分)
□ 所有伏笔都回收了吗？
□ 冲突真正解决了吗？
□ 有意外感吗？
□ 同时也有"原来如此"感吗？

评分: ___/10

总分: ___/40
合格线: 30分以上
```

### 从高潮倒推铺垫

```
高潮设定: [描述高潮场景]
    │
    ├─ 需要什么前提条件？
    │   └─ 条件A: [描述] → 需要在 S_ 埋设
    │   └─ 条件B: [描述] → 需要在 S_ 埋设
    │
    ├─ 角色需要什么状态？
    │   └─ 主角: [心理状态] → 需要 [成长历程]
    │   └─ 对手: [心理状态] → 需要 [动机铺垫]
    │
    ├─ 读者需要知道什么？
    │   └─ 信息A: [描述] → 在 S_ 透露
    │   └─ 信息B: [描述] → 在 S_ 透露
    │
    └─ 情感需要什么铺垫？
        └─ [情感] → 需要 [场景/事件]
```

## 节奏控制指南

### 节奏与格子的关系

| 节奏 | 格子/页 | 格子大小 | 对话量 | 效果线 | 适用场景 |
|------|---------|----------|--------|--------|----------|
| **极快** | 7-9 | 小格为主 | 最少 | 速度线 | 战斗、追逐 |
| **快** | 5-7 | 小格+中格 | 少 | 集中线 | 动作、紧张 |
| **中** | 4-5 | 中格为主 | 中等 | 适度 | 对话、日常 |
| **慢** | 3-4 | 中格+大格 | 多 | 少 | 情感、思考 |
| **极慢** | 1-2 | 大格/跨页 | 最少 | 氛围效果 | 高潮、震撼 |

### 节奏变化原则

```
错误: ─────────────────────────── (平坦无变化)

正确: ──╱──╲─╱╱╱──╲──╱╱╱╱──╲── (有起伏)
         ↑   ↑↑↑  ↑   ↑↑↑↑
         小  动作  转  高潮
         转  加速  折  爆发
         折
```

**变化规则**:
1. 每2-3页应有节奏变化
2. 高潮前需要"减速蓄力"
3. 高潮后需要"减速释放"
4. 连续快节奏不超过4页
5. 连续慢节奏不超过3页

### 翻页悬念 (めくり)

每一页的最后一格承担"翻页钩子"功能：

**有效钩子类型**:
1. 悬念: 「那是什么...!?」
2. 预告: 新角色的剪影/脚步声
3. 反问: 角色提出的问题
4. 动作起势: 准备攻击/准备告白
5. 情绪预告: 表情变化的开始

**无效钩子**:
- 完整的信息交代
- 情绪的完全释放
- 场景的完整结束

## 结构诊断清单

### 整体结构检查

```markdown
## 结构诊断

### 开头检查 (起/Act1)
□ 开场能在30秒内抓住注意力吗？
□ 主角在第3页前出场了吗？
□ 世界观够清晰但不过度解释吗？
□ 触发事件足够有力吗？

### 中段检查 (承转/Act2)
□ 冲突持续升级了吗？
□ 有明确的中点转折吗？
□ 角色有成长/变化吗？
□ 节奏有变化吗？

### 高潮检查 (転/Climax)
□ 高潮场景足够有冲击力吗？
□ 给予了足够的版面吗？
□ 情感爆发是铺垫充分的吗？
□ 解决方式合理吗？

### 结尾检查 (結/Act3)
□ 主题得到回应了吗？
□ 伏笔都回收了吗？
□ 角色弧线完整吗？
□ 留有余韵吗？
```

### 常见结构问题

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| 开头太慢 | 读者第3页前失去兴趣 | 从动作/冲突开始，背景融入后续 |
| 中段拖沓 | 感觉"什么都没发生" | 增加子冲突，每场景都有变化 |
| 转折太弱 | "就这样？" | 提高赌注，增加意外性 |
| 高潮太短 | 感觉匆忙 | 扩展高潮，至少占总页数的15% |
| 结尾仓促 | 戛然而止 | 增加呼应开头的场景 |
| 结尾拖沓 | 感觉啰嗦 | 删除不必要的解释 |
