# 角色设定系统 (Character System)

本文档是角色创建的完整指南，是 SKILL.md Phase 2 的深度参考。涵盖心理架构、原型库、关系设计和视觉规范。

---

## 角色心理架构

### Want/Need/Flaw 框架

这是角色深度的核心框架，所有主要角色都应该有这三层设计。

```
┌─────────────────────────────────────────────────────────────┐
│                        WANT (表面欲望)                       │
│            角色自认为想要的东西，驱动外部行动                   │
├─────────────────────────────────────────────────────────────┤
│                        NEED (内心需要)                       │
│            角色真正需要的东西，通常自己不知道                   │
├─────────────────────────────────────────────────────────────┤
│                        FLAW (性格缺陷)                       │
│            阻止角色获得NEED的内在障碍                          │
├─────────────────────────────────────────────────────────────┤
│                        GHOST (过去创伤)                      │
│            造成FLAW的根源，角色背负的过去                      │
└─────────────────────────────────────────────────────────────┘
```

### 框架详解

**WANT (表面欲望)**:
- 角色在故事开始时明确追求的目标
- 通常是外在的、可量化的
- 驱动剧情前进
- 例: 变强、打败敌人、得到某物、证明自己

**NEED (内心需要)**:
- 角色真正需要的东西
- 通常是内在的、情感的
- 角色常常不自知
- 故事的主题往往与此相关
- 例: 接受帮助、学会放手、原谅自己、真正的连接

**FLAW (性格缺陷)**:
- 阻止角色获得NEED的内在障碍
- 是角色弧线的核心
- 需要在故事中被克服（或加深，悲剧向）
- 例: 过度逞强、无法信任、恐惧亲密、自我否定

**GHOST (过去创伤)**:
- 造成FLAW的根源事件
- 可以直接展示或暗示
- 是角色深度的来源
- 例: 童年被抛弃、失败的经历、重要的人的死亡

### 框架应用示例

**少年漫画主角示例**:
```yaml
character: 热血少年主角

want: "想要变强，打败敌人，保护重要的人"
need: "接受帮助，学会依靠伙伴"
flaw: "过度逞强，拒绝示弱"
ghost: "童年被欺负，发誓不再软弱"

# 内部冲突: 想独自承担 vs 需要团队合作
# 角色弧线: 从独狼到信任伙伴的旅程
# 高潮设计: 在不得不接受帮助的情况下获胜
```

**少女漫画女主角示例**:
```yaml
character: 平凡少女主角

want: "得到暗恋对象的爱"
need: "先爱自己，确认自我价值"
flaw: "不够自信，总觉得自己配不上"
ghost: "从小被比较，总是'不够好'"

# 内部冲突: 追求外部认可 vs 需要内部肯定
# 角色弧线: 从依赖他人评价到自我接纳
# 高潮设计: 告白的同时确认了自己的价值
```

**反派示例**:
```yaml
character: 悲剧反派

want: "毁灭/复仇/统治"
need: "被理解，被接纳"
flaw: "认为暴力是唯一解决方式"
ghost: "被这个世界背叛/抛弃"

# 内部冲突: 破坏欲 vs 渴望被理解
# 角色弧线: 揭示其悲剧根源，提供理解的可能
# 高潮设计: 在被理解的瞬间选择（救赎或毁灭）
```

## 角色档案模板

### 完整档案模板

```yaml
# ============================================================
# 角色档案 - [角色名]
# ============================================================

# ======================== 身份层 ========================
character_id: "[类型]_[编号]"  # hero_001, heroine_001, rival_001
name: "[中文名]"
name_reading: "[日语读音]"  # 可选
nickname: "[昵称/外号]"  # 可选
role: "[故事角色]"  # 主角、女主角、对手、伙伴、导师
archetype: "[角色原型]"  # 见原型库

# ======================== 心理层 ========================
# Want/Need/Flaw 框架
psychology:
  want: "[表面欲望]"
  need: "[内心需要]"
  flaw: "[性格缺陷]"
  ghost: "[过去创伤]"

  # 冲突来源
  internal_conflict: "[want vs need 的冲突]"
  external_conflict: "[阻止 want 的外部障碍]"

# ======================== 表达层 ========================
personality:
  traits:
    - "[性格特点1]"
    - "[性格特点2]"
    - "[性格特点3]"

  speech_pattern: "[说话风格]"
  # 热血/冷静/毒舌/元气/敬语/方言...

  signature_phrase: "[口头禅/标志性台词]"
  # 例: 「绝对不会放弃！」

  emotional_range: "[情绪表达范围]"
  # 热血型: 决心、愤怒、惊讶、友情感动、搞笑
  # 温柔型: 微笑、害羞、惊讶、感动、难过

# ======================== 视觉层 ========================
visual:
  # 视觉原型
  visual_archetype: "[视觉原型]"
  # 少年: 热血锐眼型/天才冷酷型/成长温和型
  # 少女: 大眼萌系/清秀美人/元气短发

  # 面部特征
  face:
    cn: "[中文描述]"
    en: "[English description]"

  # 体型
  build:
    cn: "[中文描述]"
    en: "[English description]"

  # 服装
  clothing:
    default:
      cn: "[默认服装-中文]"
      en: "[Default outfit-English]"
    casual:
      cn: "[便装-中文]"
      en: "[Casual-English]"
    special:
      cn: "[特殊服装-中文]"
      en: "[Special-English]"

  # 标志性特征（重要！角色辨识）
  distinctive_features:
    cn: "[独特记号-中文]"
    en: "[Distinctive features-English]"

  # 表情库
  expression_library:
    - "[表情1]: [描述]"
    - "[表情2]: [描述]"
    - "[表情3]: [描述]"

# ======================== 日漫视觉风格 ========================
manga_visual:
  eye_style: "[眼型描述]"
  # 少年漫画: 锐利型、坚定型、热血型
  # 少女漫画: 大眼睛型、闪亮型、温柔型

  hair_style: "[发型描述]"
  # 动态特性: 飘动、刺猬、双马尾...

  common_effects:
    - "[效果1]"  # 如: 汗滴、愤怒青筋
    - "[效果2]"

# ======================== 关系层 ========================
relationships:
  - target: "[角色ID]"
    type: "[关系类型]"
    dynamic: "[关系动态]"
    chemistry_potential: "[化学反应潜力]"

# ======================== 角色弧线 ========================
character_arc:
  arc_type: "[弧线类型]"  # 成长弧/堕落弧/平坦弧
  starting_state: "[开始状态]"
  ending_state: "[结束状态]"
  key_transformation: "[关键转变]"
  transformation_trigger: "[触发转变的事件]"

# ======================== AI 参考（可选）========================
ai_reference:
  cref_url: "[Character Reference URL]"
  cw: 85
  oref_url: "[Object Reference URL]"
  style_keywords: "manga style, black and white, [shounen/shoujo]"
```

## 角色原型库

### 少年漫画角色原型

#### 热血型主角

```yaml
archetype: "热血型主角"

psychology:
  want: "变强/成为最强/保护重要的人"
  need: "学会依靠伙伴/接受自己的弱点"
  flaw: "冲动/过度逞强/不擅动脑"
  ghost: "童年弱小的记忆/重要的人的牺牲"

personality:
  traits: ["正义感强", "重视伙伴", "永不放弃", "单纯直接"]
  speech_pattern: "热血、直接、喜欢喊必杀技名"
  signature_phrase: "「绝对不会放弃！」「我要保护你！」"

manga_visual:
  eye_style: "锐利型，充满决心，瞳孔较小"
  expression_range: "决心、愤怒、惊讶、友情感动、搞笑脸"
  common_effects: ["热血背景", "集中线", "青筋"]

weakness: "冲动、不擅长动脑、容易被激将"
arc_type: "成长弧 - 从独狼到领袖"
```

**代表角色**: 孙悟空《龙珠》、路飞《海贼王》、鸣人《火影忍者》

#### 天才型主角

```yaml
archetype: "天才型主角"

psychology:
  want: "证明自己/解决谜题/改变世界"
  need: "真正的人际连接/承认自己的局限"
  flaw: "傲慢/社交障碍/过度理性"
  ghost: "不被理解的天才童年/能力的诅咒"

personality:
  traits: ["聪明绝顶", "冷静分析", "傲慢或内敛", "有原则"]
  speech_pattern: "冷静、分析性、偶尔毒舌"
  signature_phrase: "「计划通り」「真相只有一个」"

manga_visual:
  eye_style: "锐利型，洞察一切"
  expression_range: "冷笑、分析、惊讶（少见）、决心"
  common_effects: ["黑暗背景", "智慧光芒", "棋盘意象"]

weakness: "过度自信、人际关系、情感盲点"
arc_type: "成长弧 - 从孤高到接纳 / 堕落弧 - 走向毁灭"
```

**代表角色**: 夜神月《死亡笔记》、L《死亡笔记》、工藤新一《柯南》

#### 成长型主角

```yaml
archetype: "成长型主角"

psychology:
  want: "保护家人/完成使命/变得更强"
  need: "接受失去/找到自己的道路"
  flaw: "初期能力不足/过于善良/优柔寡断"
  ghost: "失去重要的人/平凡的出身"

personality:
  traits: ["善良", "坚韧", "不断成长", "重视家人"]
  speech_pattern: "温和但坚定，真诚"
  signature_phrase: "「我会变得更强！」「绝不让悲剧重演」"

manga_visual:
  eye_style: "温和但有力"
  expression_range: "善良、决心、悲伤、愤怒、成长蜕变"
  common_effects: ["成长光芒", "回忆闪回"]

weakness: "初期能力不足、过于善良、背负太多"
arc_type: "成长弧 - 从弱小到强大"
```

**代表角色**: 炭治郎《鬼灭之刃》、绿谷出久《我的英雄学院》

#### 宿敌/对手

```yaml
archetype: "宿敌型"

psychology:
  want: "超越主角/证明自己/获得认可"
  need: "放下骄傲/真正的羁绊"
  flaw: "骄傲/孤独/无法认输"
  ghost: "被主角超越/不被认可的过去"

personality:
  traits: ["骄傲", "实力强", "有自己的正义", "不服输"]
  speech_pattern: "傲慢、挑衅、后期可能温和"
  signature_phrase: "「我才是最强的！」「下次绝对打败你」"

manga_visual:
  eye_style: "锐利、挑衅"
  expression_range: "傲慢、愤怒、惊讶、(隐藏的)尊重"
  common_effects: ["对峙分割线", "力量对比"]

weakness: "骄傲、孤独、无法坦诚"
arc_type: "成长弧 - 从对手到战友"
```

**代表角色**: 贝吉塔《龙珠》、佐助《火影忍者》、爆豪《我英》

#### 导师

```yaml
archetype: "导师型"

psychology:
  want: "传承/保护弟子/完成使命"
  need: "放手让弟子成长/面对自己的过去"
  flaw: "过度保护/未解决的遗憾"
  ghost: "失败的过去/失去的弟子"

personality:
  traits: ["睿智", "神秘", "有时严厉", "深藏温柔"]
  speech_pattern: "沉稳、点到为止、偶尔玩笑"
  signature_phrase: "「答案要自己找」「你已经超越我了」"

manga_visual:
  eye_style: "深邃、看透一切"
  expression_range: "沉稳、严肃、温和微笑、战斗时认真"
  common_effects: ["智慧光芒", "回忆画面"]

weakness: "过去的遗憾、过度保护"
arc_type: "平坦弧（引导者） / 牺牲弧"
```

**代表角色**: 卡卡西《火影》、安西教练《灌篮高手》、全真《JoJo》

| 类型 | 特征 | 适用关系 | 张力来源 |
|------|------|----------|----------|
| **互补型** | 性格相反但互相需要 | 伙伴、恋人 | 差异带来的摩擦与成长 |
| **镜像型** | 相似但选择不同 | 宿敌、兄弟 | "如果我走了另一条路" |
| **保护型** | 一方守护另一方 | 师徒、恋人 | 被保护者的成长欲望 |
| **竞争型** | 互相追赶 | 对手、朋友 | 谁更强的持续张力 |
| **误解型** | 错误认知对方 | 恋人、敌人 | 真相揭露的期待 |
| **羁绊型** | 有深厚历史 | 青梅竹马 | 过去与现在的张力 |

## 角色弧线设计

### 弧线类型

**成长弧 (Positive Arc)**:
```
开始: 角色有FLAW，追求WANT
中间: 遭遇困难，FLAW阻碍进展
转折: 开始意识到NEED
高潮: 克服FLAW，获得NEED
结束: 成为更好的自己
```

**堕落弧 (Negative Arc)**:
```
开始: 角色有某种信念
中间: 信念受到挑战
转折: 选择了错误的道路
高潮: 信念崩塌或走向极端
结束: 成为更糟的自己/毁灭
```

**平坦弧 (Flat Arc)**:
```
开始: 角色有正确的信念
中间: 世界挑战这个信念
过程: 角色改变周围的人/世界
结束: 信念经过考验更加坚定
注: 角色本身不变，但改变他人
```

### 弧线设计模板

```markdown
## 角色弧线: [角色名]

### 弧线类型
[成长弧/堕落弧/平坦弧]

### 状态变化
- 开始状态: [描述FLAW主导的状态]
- 结束状态: [描述克服FLAW后的状态]

### 关键节点
1. **触发点** (P___): [第一次认识到问题]
2. **尝试改变** (P___): [第一次尝试但失败]
3. **最低点** (P___): [FLAW导致最大危机]
4. **领悟时刻** (P___): [真正理解NEED]
5. **转变时刻** (P___): [做出改变的行动]
6. **证明时刻** (P___): [在高潮中展示改变]

### 外在表现
变化如何通过以下方面展现：
- 行为: [前后对比]
- 台词: [前后对比]
- 表情: [前后对比]
- 与他人的关系: [前后对比]
```
