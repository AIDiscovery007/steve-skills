# 黄金分析指标参考

## 核心三要素（按重要性排序）

### 1. 实际利率（TIP ETF）— 最强结构性驱动
- **逻辑**：黄金无息，实际利率越低，持有黄金的机会成本越低
- **信号**：TIP 价格与实际利率**反向**
  - TIP 涨 → 实际利率降 → 黄金受益
  - TIP 跌 → 实际利率升 → 黄金承压
- **阈值**：TIP 在52周区间 >70% = 绿灯；40-70% = 黄灯；<40% = 红灯

### 2. 美元指数（DXY）— 最强短期变量
- **逻辑**：黄金以美元计价，美元升值直接压低以其他货币计算的金价
- **信号**：DXY 与黄金**负相关**
  - DXY 跌 → 黄金涨（其他条件相同）
  - DXY 涨 → 黄金承压
- **关键位**：100 整数关口是心理支撑/压力；突破 101 需警惕

### 3. 名义利率（TYX 30年）— 通胀预期参考
- **用途**：TYX - 实际利率 ≈ 盈亏平衡通胀率（BEI）
- **逻辑**：TYX 高 → 长端融资成本高 → 不利长久期资产
- **阈值**：>5.0% 红灯；4.5-5.0% 黄灯；<4.5% 绿灯

## 流动性风险指标（辅助过滤）

### MOVE 指数 — 利率波动温度计
- **作用**：黄金仓位的风险环境过滤器
- **机制**：
  - MOVE 高 → 机构承压去杠杆 → 黄金可能被抛售（非因黄金本身）
  - MOVE 低 → 市场平静 → 趋势性逻辑可以更放心跟
- **阈值**：>130 极高风险；100-130 偏高；80-100 温和；<80 正常
- **注意**：MOVE 与黄金相关性不稳定，不可单独用作交易信号

### USDJPY — 套息交易压力
- **机制**：日元快速升值（USDJPY 急跌）→ carry trade 平仓 → 全市场去杠杆 → 黄金短期下跌
- **信号阈值**：单日跌幅 > -2% 极高风险；> -1% 需关注
- **背景**：USDJPY 高位（>155）时，反转风险更需要盯紧

### HYG — 信用市场健康度
- **作用**：确认融资环境是否正常
- **信号**：HYG 从高位大幅下跌（>5%）= 信用利差走阔 = 融资端收紧 = 黄金承压
- **阈值**：从52周高点回撤 >5% 红灯；>2% 黄灯

## 综合评分逻辑

```
绿灯 × 15 + 黄灯 × 7 + 红灯 × (-20) → 0-100分
```

| 分数 | 建议 |
|------|------|
| ≥70  | 积极做多 |
| 50-70 | 谨慎持有 |
| 30-50 | 观望为主 |
| <30  | 规避/减仓 |

## 金银比参考

金银比 = 黄金价格 / 白银价格

| 金银比 | 含义 |
|--------|------|
| >80    | 白银相对低估，黄金相对高估；通常在经济不确定时出现 |
| 60-80  | 历史正常区间 |
| <60    | 白银相对高估，工业需求旺盛，风险偏好高 |

金银比高 → 如果看多贵金属，白银弹性更大（但波动也更大）
