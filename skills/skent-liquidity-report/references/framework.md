# 流动性分析框架

## 核心逻辑链

```
净流动性水位下降（ON RRP 余额高 / TGA 上升）
    ↓
SOFR 上升（短端钱贵，资金紧张）
    ↓
MOVE 上升（利率波动大，杠杆撑不住，中介缩表）
    ↓
USDJPY 急跌（日元升值，套息交易 carry trade 平仓）
    ↓
HYG 下跌（高收益债利差走阔，融资端收紧确认）
    ↓
CME 提高保证金 → 强制去杠杆 → 风险资产踩踏
```

## 指标解读

### 1. 净流动性（ON RRP 作为代理）
- 公式：净流动性 = 美联储总资产 - TGA（财政部账户）- ON RRP（隔夜逆回购）
- ON RRP 余额低 → 流动性已流入市场 → 偏宽松
- ON RRP 余额高 → 流动性被"锁住" → 偏收紧

### 2. SOFR
- 隔夜担保融资利率，美元短端流动性的实时温度计
- SOFR 异常上升 → 短端资金紧张 → 触发资产抛售/偿还杠杆连锁

### 3. MOVE 指数
- 美债市场波动率，类似股市的 VIX
- MOVE 上行 → 利率波动变大 → 债券交易商缩表 → 杠杆下降 → 风险资产承压
- CME 保证金与 MOVE 高度相关：MOVE 飙升时 CME 往往提高保证金要求

### 4. USDJPY + 套息交易
- 日元是全球最大套息交易（carry trade）融资货币
- USDJPY 急跌（日元升值）→ 套息收益消失 → 被迫平仓 → 抛售全球风险资产
- 2024年8月闪崩案例：USDJPY 从160跌至142，触发全球股市暴跌

### 5. HYG（高收益债 ETF）
- 价格下跌 = 高收益债收益率上升 = 信用利差走阔
- 信用利差走阔 = 融资成本上升 = 高杠杆企业融资压力加大
- 作为流动性危机"最终确认信号"

### 6. CME 保证金机制
- 极端波动时，CME 提高期货/衍生品保证金要求
- 强制所有持有人追加保证金或平仓 → 去杠杆加速
- 历史案例：
  - 2011年白银暴跌：保证金从11% → 18%，价格从50跌至33
  - 2020年原油负价：CME 连续提高保证金
  - 每次提高保证金 = 市场踩踏信号

| MOVE | < 100 | 100-130 | > 130 |

| HYG 从52周高位回撤 | < -2% | -2% ~ -5% | < -5% |
| ON RRP | < $100B | $100-500B | > $500B |
