# 黄金分析指标参考

## 核心三要素（按重要性排序）

### 1. 实际利率（TIP ETF）— 最强结构性驱动
- **逻辑**：黄金无息，实际利率越低，持有黄金的机会成本越低
- **信号**：TIP 价格与实际利率**反向**
  - TIP 涨 → 实际利率降 → 黄金受益
  - TIP 跌 → 实际利率升 → 黄金承压
- **阈值**：TIP 在52周区间 >70% = 绿灯；40-70% = 黄灯；<40% = 红灯

### 2. 美元指数（DXY）— 最强短期变量
- **逻辑**：黄金以美元计价，美元升值直接压低以其他货币计算的金价
- **信号**：DXY 与黄金**负相关**
  - DXY 跌 → 黄金涨（其他条件相同）
  - DXY 涨 → 黄金承压
- **关键位**：100 整数关口是心理支撑/压力；突破 101 需警惕

### 3. 名义利率（TYX 30年）— 通胀预期参考
- **用途**：TYX - 实际利率 ≈ 盈亏平衡通胀率（BEI）
- **逻辑**：TYX 高 → 长端融资成本高 → 不利长久期资产
- **阈值**：>5.0% 红灯；4.5-5.0% 黄灯；<4.5% 绿灯

## 流动性风险指标（辅助过滤）

### MOVE 指数 — 利率波动温度计
- **作用**：黄金仓位的风险环境过滤器
- **机制**：
  - MOVE 高 → 机构承压去杠杆 → 黄金可能被抛售（非因黄金本身）
  - MOVE 低 → 市场平静 → 趋势性逻辑可以更放心跟
- **阈值**：>130 极高风险；100-130 偏高；80-100 温和；<80 正常
- **注意**：MOVE 与黄金相关性不稳定，不可单独用作交易信号

### USDJPY — 套息交易压力
- **机制**：日元快速升值（USDJPY 急跌）→ carry trade 平仓 → 全市场去杠杆 → 黄金短期下跌
- **信号阈值**：单日跌幅 > -2% 极高风险；> -1% 需关注
- **背景**：USDJPY 高位（>155）时，反转风险更需要盯紧

### HYG — 信用市场健康度
- **作用**：确认融资环境是否正常
- **信号**：HYG 从高位大幅下跌（>5%）= 信用利差走阔 = 融资端收紧 = 黄金承压
- **阈值**：从52周高点回撤 >5% 红灯；>2% 黄灯

## 金银比参考

金银比 = 黄金价格 / 白银价格

| 金银比 | 含义 |
|--------|------|
| >80    | 白银相对低估，黄金相对高估；通常在经济不确定时出现 |
| 60-80  | 历史正常区间 |
| <60    | 白银相对高估，工业需求旺盛，风险偏好高 |

金银比高 → 如果看多贵金属，白银弹性更大（但波动也更大）

## 核心理论：实际利率是黄金唯一定价因子

### 基本公式
```
TIPS收益率 = 美国国债收益率 - 通胀预期（Breakeven）+ 残差
```

黄金不产生利息。当实际利率接近 2% 时，持有美债的真实回报（抗通胀后的收益）已经非常有吸引力。

### 油价 → 黄金的完整传导链

**触发路径**：
```
霍尔木兹海峡风险 → 油价冲破 $100
    ↓
通胀预期（Breakeven）上升
    ↓
但市场更担心联储维持 Higher for Longer
    ↓
名义利率涨得比通胀预期更快
    ↓
实际利率走高（这是关键！）
    ↓
黄金承压
```

### 关键洞察
1. **油价上涨 ≠ 直接利好黄金** — 取决于通胀预期与名义利率谁跑得更快
2. **实际利率才是唯一裁判** — 当名义利率上升幅度超过通胀预期，实际利率走高，黄金反而下跌
3. **"Higher for Longer"陷阱** — 若市场定价联储不会降息甚至可能加息，长端名义利率上行，即使油价带来通胀预期，实际利率仍可能上升压制黄金
4. **残差项** — TIPS收益率公式中的"残差"代表市场对未來不确定性的定价，在地缘危机时为正（利好黄金），但可能被名义利率涨幅抵消

### 实际案例（2024-2026）
- 油价 $100+ 且实际利率 >2% 时，黄金表现疲软（2024年Q3）
- 油价 $80-90 但降息预期升温时，黄金大涨（2024年Q4）
- 结论：**做多黄金，本质是做空实际利率**

### 实战意义
- 当黄金与油价同向下跌时，说明市场在交易"滞胀"或"更高利率"预期
- 当黄金与油价背离（油价涨，黄金跌），说明名义利率上行预期占主导，是黄金最危险的信号
- 监测指标：`TYX - TIP` 价差走阔 → 实际利率上行压力 → 黄金需警惕
