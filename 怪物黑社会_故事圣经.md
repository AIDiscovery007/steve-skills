# 故事圣经：《怪物黑社会》

> 黄昏时分，人与怪物的交界——有一群怪物，以暴制暴，守护弱小。

---

## 执行摘要

| 项目 | 内容 |
|------|------|
| **类型** | 四格漫画 / 暗黑正义 + 治愈喜剧 |
| **一句话概念** | 潜伏在黑暗中的怪物黑社会，用黑道方式守护弱小 |
| **核心主题** | 外表与内心的反差；真正的"怪物"是谁？ |
| **情感基调** | 硬核燃向 + 会心一笑的治愈感 |
| **目标读者** | 喜欢反差萌、暗黑正义、怪物题材的读者 |
| **参考作品** | 《地狱少女》的暗黑正义感 |
| **格式** | 四格漫画（可发展为系列） |

## 世界观设定

### 基础设定

```yaml
世界类型: 现代都市 + 怪物共存
时代背景: 现代日本/架空都市
核心设定: 怪物与人类共存，但关系紧张

人怪关系:
  主流: 人类歧视/恐惧怪物
  例外: 少数人与怪物和谐共处
  关键: 「黄昏亭」是人怪交界的中立地带
```

### 怪物社会

```yaml
怪物黑社会:
  名称: 未正式命名（可称"黄昏组"/"达斯克家族"）
  据点: 「黄昏亭」居酒屋
  势力范围: 城市暗面

组织规矩:
  - 不许欺负弱小（无论人类还是怪物）
  - 不许给怪物丢脸（作恶的坏怪物要收拾）
  - 以暴制暴，但只针对恶人

行事风格:
  - 黑道做派，言出必行
  - 暗中行动，不求名声
  - 保护被欺负的人类和善良的怪物
```

### 「黄昏亭」

```yaml
类型: 居酒屋
位置: 城市边缘的小巷深处
外观: 老旧日式建筑，门口挂着昏黄灯笼
氛围: 温暖、神秘、像是时间静止的地方

店名含义:
  "黄昏"——日与夜的交界
  象征人类与怪物共存的可能性

常客:
  - 怪物黑社会成员
  - 知道真相的少数人类
  - 偶尔误入的普通人（会被"处理记忆"？）

特色:
  - 银牙亲手做的料理
  - 组长专属的"番茄汁"（假装是血）
  - 老爷爷什么都知道但什么都不说
```

### 角色1：奥德里克·达斯克 (Aldric Dusk)

**吸血鬼组长 | 黄昏的古老统治者**

#### 基础信息

```yaml
身份: 怪物黑社会组长
种族: 吸血鬼（纯血贵族）
年龄: 300岁以上
身高: 187cm

名字含义:
  Aldric = 古老的统治者（日耳曼语源）
  Dusk = 黄昏
  → "黄昏的古老统治者"
```

#### 外表特征

```yaml
整体印象: 黑道贵族，压迫感十足

面部:
  - 苍白肌肤，如同大理石雕塑
  - 猩红色眼眸，黑暗中发光
  - 锐利的獠牙，平时隐藏
  - 高挺鼻梁，轮廓分明
  - 表情永远冷酷，极少变化

发型: 黑色短发，梳理整齐，一丝不苟

服装:
  - 三件套黑色西装（标配）
  - 银色袖扣，低调奢华
  - 黑色长风衣（外出时）
  - 永远穿得一丝不苟

特征:
  - 左手无名指戴着古老的家族戒指
  - 站姿永远笔直，贵族气质
```

#### 性格层次

```yaml
# Want/Need/Flaw 框架

表面人设 (Want):
  - 冷酷、威严、寡言
  - "一个眼神让你腿软"
  - 黑道老大的绝对气场
  - 说话简洁，从不废话

内心真实 (Need):
  - 极度怕孤独
  - 渴望被需要、被接纳
  - 其实很珍惜身边的人

致命弱点 (Flaw):
  - 对小动物完全没辙（特别是猫）
  - 对小孩子束手无策
  - 被真心感谢时会石化
  - 死也不承认自己的软肋

过去创伤 (Ghost):
  - 数百年的孤独
  - 曾被人类背叛？（可深挖）
  - 因此建立了怪物黑社会，保护同类
```

#### 反差名场面

```yaml
冷酷时刻:
  - "在我的地盘，没有我不知道的事。"
  - "滚。下次让我看到，你会知道什么是真正的怪物。"
  - 单手拎起恶人，面无表情

破功时刻:
  - 被流浪猫蹭腿 → 完全石化不敢动
  - 被小孩抱住 → 手不知道往哪放
  - 被叫"吸血鬼叔叔好帅" → 耳朵红但脸上还装酷
  - 深夜在黄昏亭喝番茄汁假装是血
```

#### 称呼体系

```yaml
自称: 本大爷（对外）/ 我（私下）
被称呼:
  - 正式: 达斯克大人
  - 银牙: 老蝙蝠 / 阿德
  - 老爷爷: 小达（让他很无奈）
  - 被救的小孩: 吸血鬼叔叔（内心崩溃）
```

### 角色2：银牙 (Ginga)

**狼人副手 | 料理与战斗的双面狼**

#### 基础信息

```yaml
身份: 组长的搭档兼损友
种族: 狼人
年龄: 120岁
身高: 182cm

名字含义:
  银牙 = 银色的獠牙
  象征狼人的野性与锋利
```

#### 外表特征

```yaml
整体印象: 野兽般的锐利感，但有生活气息

面部:
  - 小麦色肌肤，健康有活力
  - 琥珀色眼睛，像狼一样锐利
  - 锋利的犬牙，笑起来很凶
  - 下巴有一道浅浅的疤痕

发型: 银灰色乱发，像狼毛一样蓬松

服装:
  - 日常: 黑色皮夹克 + 牛仔裤
  - 在店里: 皮夹克 + 围裙（反差萌）
  - 袖子总是卷起来，露出肌肉线条

特征:
  - 左耳有两个银色耳钉
  - 脖子上戴着狼牙项链
  - 手上有做料理留下的小伤疤
```

#### 性格层次

```yaml
# Want/Need/Flaw 框架

表面人设 (Want):
  - 凶狠、暴躁、战斗狂
  - 看起来是典型的狼人打手
  - 说话直接，容易急躁

内心真实 (Need):
  - 料理高手（真正的热爱）
  - 其实非常操心组长
  - 重视同伴，愿意为朋友拼命

致命弱点 (Flaw):
  - 毒舌停不下来
  - 表面嫌弃实际关心（傲娇狼）
  - 看到组长破功就想笑

与组长关系:
  - 互损十年的过命交情
  - 吵完架还是一起喝酒
  - "损友"是最准确的定义
```

#### 反差名场面

```yaml
凶狠时刻:
  - 露出獠牙，野兽般的笑容
  - "找死？我成全你。"
  - 战斗时瞳孔变成狼的竖瞳

反差时刻:
  - 围着围裙认真做料理
  - 对食材品质非常讲究
  - 被客人夸奖料理好吃时耳朵会动
  - 偷偷给组长的番茄汁加营养
```

#### 口头禅与台词

```yaml
吐槽组长:
  - "老蝙蝠，你又被野猫驯服了？"
  - "噗——哈哈哈你脸红了——"
  - "三百岁的吸血鬼被五岁小孩叫叔叔，笑死我了"

认真时刻:
  - "老大的事就是我的事。"
  - （下一秒）"但你能不能别丢人了！"

对料理:
  - "食材要新鲜，火候要精准，这是基本。"
  - "嫌难吃？那你自己做啊。"
```

### 角色3：黄昏亭老爷爷

**神秘店主 | 看透一切的旁观者**

#### 基础信息

```yaml
身份: 「黄昏亭」店主
种族: ？？？（可能不是普通人类）
年龄: 不明（外表70岁左右）

真实身份猜测:
  - 退隐的大妖怪？
  - 某位神明的化身？
  - 曾经的"上一代组长"？
  - 死神？（负责人怪交界的存在）
```

#### 外表特征

```yaml
整体印象: 和蔼可亲但深不可测

面部:
  - 慈祥的面容，皱纹是岁月的痕迹
  - 总是眯着眼睛，看不清眼神
  - 笑容神秘，像是什么都知道

发型: 白发，整齐地向后梳

服装:
  - 传统日式居酒屋店主装扮
  - 深色和服或作务衣
  - 系着白色围裙

特征:
  - 永远在擦杯子
  - 动作从容，不急不躁
  - 偶尔说出意味深长的话
```

#### 性格与作用

```yaml
性格:
  - 温和、从容、洞察一切
  - 对组长和银牙像看孙子
  - 话不多，但每句话都有深意

故事作用:
  - 据点提供者（黄昏亭是安全屋）
  - 情报来源（"听说东区最近不太平呢~"）
  - 点破傲娇（"小达又被小朋友叫叔叔了吗？"）
  - 神秘感担当（真实身份是伏笔）

与两人关系:
  - 叫组长"小达"（组长无奈但不反驳）
  - 叫银牙"小银"
  - 看着他们互损，笑眯眯不说话
```

#### 经典台词

```yaml
日常:
  - "哎呀，小达今天又被小朋友叫叔叔了吗？来，番茄汁请客。"
  - "年轻真好啊~"（对着300岁的吸血鬼说）

提供情报:
  - "听说东区最近不太平呢......有人在欺负流浪猫哦。"
  - "那个巷子......有迷路的小孩子呢。"

意味深长:
  - "黄昏是个好时辰，不是白天也不是黑夜......谁都可以存在的时间。"
  - "怪物和人类，有什么区别呢？"
```

## 第一话剧情

### 「黄昏亭的规矩」

**四格类型**: 以暴制暴型（模式B）
**情感弧线**: 压抑 → 恐惧 → 爽快 → 会心一笑

---

#### 第1格【起】建立危机

```yaml
构图: 低角度仰视，压迫感
场景: 昏暗小巷，墙壁斑驳，一盏路灯昏黄
时间: 黄昏，天色暗沉

人物:
  - 人贩子A: 画面左侧，粗壮，抓着小女孩手臂
  - 人贩子B: 画面右侧，瘦高，警戒地看着巷口
  - 小女孩: 画面中央偏下，5-6岁，背着小书包，眼泪汪汪

台词:
  人贩子A: "别叫！再叫把你舌头割了！"
  小女孩: "唔唔......"
  旁白: "黄昏时分——人与怪物的交界时刻"

氛围: 压抑、危险、令人愤怒
技法: 暗色调网点，路灯高光处理
```

#### 第2格【承】威压登场

```yaml
构图: 人贩子背后视角，两道黑影逆光而立
场景: 巷口方向，夕阳余晖形成剪影

人物:
  - 奥德里克: 右侧剪影，西装，猩红眼眸发光
  - 银牙: 左侧剪影，皮夹克，琥珀眼睛，露出尖牙
  - 人贩子: 画面前景背影，开始发抖

台词:
  奥德里克: "在我的地盘——"
  银牙: "对小孩动手？"

氛围: 恐怖片即视感、压迫感爆表
技法: 大面积黑色剪影+眼睛高光，速度线放射
```

#### 第3格【転】以暴制暴

```yaml
构图: 动态斜角，冲击力
场景: 同一小巷

人物:
  - 奥德里克: 单手拎起人贩子A衣领，面无表情俯视
            另一只手护在小女孩身前
  - 银牙: 踩着趴在地上的人贩子B
  - 小女孩: 躲在组长身后，眼睛亮晶晶

台词:
  奥德里克: "滚。下次再让我看到——"
           "你们会知道什么是真正的怪物。"
  人贩子A: "鬼、鬼啊啊啊——！！"

氛围: 爽快、暗黑正义、帅气
技法: 背景全黑+集中线，眼睛强调处理
```

#### 第4格【結】反差萌爆发

```yaml
构图: 温馨日常感，强烈反差
场景: 同一小巷，氛围完全改变

人物:
  - 奥德里克: 完全石化，手悬在空中不知道放哪
            耳尖泛红，脸上还装冷酷
  - 小女孩: 紧紧抱着组长的腿，笑容灿烂，星星眼
  - 银牙: 捂嘴憋笑，肩膀在抖

台词:
  小女孩: "谢谢你！吸血鬼叔叔！你好帅！长大我要嫁给你！"
  奥德里克: "............"
  银牙: "噗——哈哈哈老蝙蝠你脸红了——"

旁白: "「黄昏亭」的规矩：不许欺负弱小。"
      "违者——会遇到比人类更可怕的东西。"

氛围: 会心一笑、温暖治愈、反差萌
技法: 背景加花朵星星效果，整体色调变暖
```

## 视觉风格指南

### 整体风格

```yaml
基调: 暗黑 + 治愈的反差风格
线条: 锐利为主，温情时刻变柔和
配色:
  - 冷色调为主（蓝、紫、黑）
  - 温情时刻加入暖色（橙、粉）
  - 组长的红眼睛是视觉锚点
```

### 分格对比

```yaml
紧张/战斗场景:
  - 暗色调、阴影重、压迫感
  - 锐利线条、速度线、集中线
  - 背景简化或全黑

日常/喜剧场景:
  - 明亮、加入搞笑元素
  - 少女漫画式的花朵、星星效果
  - 背景可爱化
```

### 角色作画要点

```yaml
奥德里克:
  冷酷时: 锐利眼神、阴影遮半脸、线条硬朗
  破功时: 表情几乎不变，但耳朵红、身体僵硬、出冷汗

银牙:
  战斗时: 野兽般的笑容、獠牙强调、瞳孔变竖
  吐槽时: 憋笑到扭曲、损友脸、肩膀抖动

老爷爷:
  永远: 眯眯眼、神秘微笑、从容淡定
```
